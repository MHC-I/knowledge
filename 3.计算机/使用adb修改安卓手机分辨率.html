使用adb修改安卓手机分辨率

title: 使用adb修改安卓手机分辨率
date: 2021-01-18 17:49:02
categories: 计算机
tags:

  • Android

  • ADB

1.前提条件：①电脑已经安装adb；②手机连接电脑，并且已经开启‘开发人员调试工具’

2.代码如下

1.修改为指定分辨率
#分辩率中的乘号即为小写字母x
adb shell wm size 1080x1920
adb shell wm size 1080x2340
2.恢复为默认分辨率
adb shell wm size reset
