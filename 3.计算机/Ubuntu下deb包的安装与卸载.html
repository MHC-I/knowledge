deb包的安装

在线安装

在线安装可使用apt命令

apt install [package name]

若下载速度慢或者无法连接可考虑更换国内源

离线安装

先通过cd进入deb包所在的目录，再使用dpkg命令安装

sudo dpkg -i [packge name.deb]

但是一般会报错，形如

dpkg: dependency problems prevent configuration of youdao-dict：
...
dpkg: error processing package youdao-dict (--install):
...

这时再通过下面的命令安装依赖(后面不带包名)

apt-get -f install

然后再次执行

sudo dpkg -i [packge name.deb]

这时应该就完成安装了

deb包的卸载

（完全卸载，包括配置文件和依赖，以nginx为例子）

1.删除apt缓存

sudo apt-get clean

2.卸载软件包，–purge包括配置文件

sudo apt-get --purge remove nginx

3.自动移除全部不使用的软件包

sudo apt-get autoremove

4.列出与nginx相关的软件 并删除显示的软件

dpkg --get-selections|grep nginx

sudo apt-get --purge remove nginx
sudo apt-get --purge remove nginx-common
sudo apt-get --purge remove nginx-core

5.查看是否卸载干净（卸载干净后应显示为空）

dpkg --get-selections|grep nginx
