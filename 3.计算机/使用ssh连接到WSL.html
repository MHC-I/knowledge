WSL端设置

1.安装ssh服务（Ubuntu虽然自带了，但是启动服务时会报错，仍然需要运行下面的命令重新安装）

sudo apt install openssh-server

2.修改ssh配置文件

sudo vim /etc/ssh/sshd_config #打开sshd编辑文件

13 #Port 22 -> Port [你想设置的ssh端口]                      #13行，修改sshd端口
56 #PasswordAuthentication no ->PasswordAuthentication yes #56行，允许密码登陆ssh

3.到你的服务器控制台打开刚刚设置的端口

4.进入winserver的Windows Defender防火墙打开刚刚设置的端口

5.启动ssh服务

sudo /etc/init.d/ssh stop #停止ssh服务
sudo /etc/init.d/ssh start #启动ssh服务
sudo /etc/init.d/ssh restart #重启ssh服务

ssh连接

ssh -p [刚刚设置的端口] [用户名]@[ip地址]
