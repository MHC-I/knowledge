36.阑尾疾病

  1. 1.

    解剖

    1. 1.

      阑尾起于盲肠末端，附于三条结肠带的会合点

    2. 2.

      阑尾体表投影约在脐与右髂前上棘连线中外 1/3 交界处，称为麦氏点 ( McBumey 点) ，是选择阑尾手术切口的标记点

    3. 3.

      当急性阑尾炎发病开始时，常表现为脐周的牵涉痛，属内脏性疼痛

  2. 2.

    急性阑尾炎

    1. 1.

      是最多见的急腹症

    2. 2.

      病因

      • 阑尾管腔阻塞：淋巴滤泡增生（最常见）、肠石、异物、炎性狭窄、食物残渣、蛔虫、肿瘤

      • 细菌入侵：多为 G-杆菌、厌氧菌

      • 先天畸形：阑尾过长、管腔细小

    3. 3.

      临床病理分型

      1. 1.

        急性单纯性阑尾炎

        • 病变只限于黏膜和黏膜下层

        • 大体：轻度肿胀，浆膜充血并失去正常光泽，表面有少量纤维蛋白渗出物

        • 镜下：阑尾各层均有水肿和中性 粒细胞浸润，黏膜表面有小溃疡和出血点

      2. 2.

        急性化脓性阑尾炎

        • 病变累及阑尾壁全层

        • 大体：明显肿胀、浆膜高度充血并覆以脓性渗出物

        • 镜下：溃疡面深达肌层和浆膜层，管壁各层有小脓肿形成，腔内明显积脓

        • 常由单纯性阑尾炎发展而来

      3. 3.

        坏疽性及穿孔性阑尾炎

        • 重型阑尾炎

        • 阑尾管壁坏死，呈暗紫色或黑色

        • 穿孔部位多在阑尾根部和尖端

      4. 4.

        阑尾周围脓肿：坏疽性及穿孔性阑尾炎病情进展缓慢时，穿孔的阑尾可被大网膜/邻近的肠管包裹而成

      5. 5.

        急性阑尾炎转归：炎症消退、炎症局限、炎症扩散

    4. 4.

      临床表现

      1. 1.

        症状

        1. 1.

          转移性腹痛：始于上腹，逐渐移向脐部，数小时（6-8 小时）后转移并局限在右下腹

          类型 腹痛性质
          单纯性 轻度隐痛
          化脓性 阵发性胀痛和剧痛
          坏疽性 持续性剧烈腹痛
          穿孔性 穿孔时压力骤减，腹痛可暂时减轻，但出现腹膜炎后，腹痛又会持续加剧
          位置 腹痛部位
          盲肠后位阑尾炎 右侧腰部
          盆位阑尾炎 耻骨上区
          肝下区阑尾炎 右上腹痛
          下腹部阑尾炎 左下腹部
        2. 2.

          胃肠道症状：厌食、恶心呕吐、腹泻等；盆部阑尾炎可引起里急后重

        3. 3.

          全身症状：乏力、中毒症状（特别是穿孔时；但体温升高不会早于腹痛）

      2. 2.

        体征

        1. 1.

          右下腹压痛：最常见的重要体征，常位于麦氏点，可随阑尾位置的变异而改变，但压痛点始终在一个固定的位置上

        2. 2.

          腹膜剌激征象：反跳痛( Blumberg 征)，腹肌紧张，肠呜音减弱或消失

        3. 3.

          右下腹包块：边界不清、固定、压痛 ➡️ 阑尾周围脓肿/阑尾炎性肿块

      3. 3.

        可作为辅助诊断的其他体征

        1. 1.

          结肠充气试验( Rovsing 征)：病人仰卧位，用右手压迫左下腹，再用左手挤压近侧结肠，结肠内气体可传至盲肠和阑尾，引起右下腹疼痛者为阳性。

        2. 2.

          腰大肌试验( Psoas 征)：病人左侧卧，使右大腿后伸，引起右下腹疼痛者为阳性。说明阑尾位于腰大肌前方，盲肠后位或腹膜后位。

        3. 3.

          闭孔内肌试验( Obturator 征)：病人仰卧位，使右髋和右大腿屈曲， 然后被动向内旋转，引起右下腹疼痛者为阳性。提示阑尾靠近闭孔内肌。

        4. 4.

          经肛门直肠指检：引起炎症阑尾所在位置压痛

    5. 5.

      辅助检查

      1. 1.

        实验室检查：

        • 白细胞计数升高到(10~20)×109/L，可发生核左移

        • 鉴别诊断：尿常规（一般无阳性发现，但累及尿路时也有 RBC）、血清淀粉酶和脂肪酶（除外急性胰腺炎）、hCG（除外异位妊娠）

      2. 2.

        影像学检查：平片可见盲肠扩张和液气平面；CT 敏感性优于超声，有助于阑尾周围脓肿的诊断

      3. 3.

        腹腔镜检查：对明确诊断具有决定性作用

    6. 6.

      鉴别诊断

      1. 1.

        胃十二指肠溃疡穿孔：突然发作的剧烈腹痛；腹膜刺激征；X 线、CT 膈下游离气体

      2. 2.

        右侧输尿管结石：右下腹阵发性剧烈绞痛，疼痛向会阴部、外生殖器放射；超声、X 线结石阴影

      3. 3.

        妇产科疾病

        • 异位妊娠破裂：急性失血症状和腹腔内出血的体征，有停经史及阴道不规则出血史

        • 卵巢滤泡或黄体囊肿破裂：异位妊娠相似（较轻）

        • 急性输卵管炎和急性盆腔炎：下腹痛逐渐发生；腹部压痛点较低；伴发热及白细胞计数升高

        • 卵巢囊肿蒂扭转：明显而剧烈腹痛，腹部/盆腔扪及有压痛性的肿块

      4. 4.

        急性肠系膜淋巴结炎：多见于儿童；范围不太固定且较广；超声或 CT 检查发现腹腔淋巴结肿大

    7. 7.

      治疗

      1. 1.

        治疗原则

        发病情况 处理原则
        急性单纯性阑尾炎 可先行非手术治疗，但必须仔细观察，如病情有发展应及时中转手术。经保守治疗后，可能遗留有阑尾腔的狭窄，且再次急性发作的机会很大
        化脓性、穿孔性阑尾炎 原则上应立即实施急诊手术，切除病理性阑尾，术后应积极抗感染，预防并发症
        发病已数日且合并炎性包块的阑尾炎 暂行保守治疗，促进炎症的尽快呼吸，待 3－6 个月后如仍有症状者，再考虑切除阑尾。保守期间如脓肿有扩大并可能破溃时，应急诊引流
        高龄病人，小儿及妊娠期急性阑尾炎 原则上应和成年人阑尾炎一样，急诊手术
      2. 2.

        手术方法：

        • 阑尾切除术：可选择硬膜外麻醉，右下腹麦氏切口或经右下腹直肌探查切口完成。主要方式为阑尾切除术（有常规法和逆行法）。粘连严重者也可行浆膜下切除阑尾。少数阑尾脓肿保守无效时可行切开引流，腹腔渗出多时，放置引流物。

        • 腹腔镜阑尾切除术：可诊断、可治疗

    8. 8.

      并发症

      1. 1.

        急性阑尾炎的并发症

        • 腹腔脓肿

        • 内、外瘘形成

        • 化脓性门静脉炎（pylephlebitis）

      2. 2.

        阑尾切除术后并发症

        • 出血：多因阑尾系膜止血不完善而或血管结扎线松脱所致；应立即输血并再次手术止血

        • 切口感染：最常见； 表现：术后 2~3 日体温升高，切口胀痛或跳痛， 局部红肿、压痛； 处理：先行试穿抽出脓液，或于波动处拆除缝线，排出脓液，放置引流，定期换药

        • 粘连性肠梗阻：与局部炎症重、手术损伤、切口异物、术后卧床（术后应早期离床）等有关

        • 粪瘘：阑尾残端单纯结扎，其结扎线脱落；盲肠组织水肿脆弱术中缝合时裂伤

  3. 3.

    特殊类型阑尾炎

    1. 1.

      新生儿急性阑尾炎：新生儿阑尾呈漏斗状，故少见；不能提供病史、临床症状不典型，故难以早期确诊

    2. 2.

      小儿急性阑尾炎

      • 大网膜发育不全，不能起到足够的保护作用

      • 病情发展较快且较重，早期即出现高热 、呕吐等症状

      • 右下腹体征不明显、不典型，但有局部压痛和肌紧张（是小儿阑尾炎的重要体征）

      • 穿孔率较高，并发症和死亡率也较高

    3. 3.

      妊娠期急性阑尾炎

      • 子宫推挤，压痛上移

      • 炎症阑尾刺激不到壁腹膜，压痛、肌紧张、反跳痛均不明显

      • 大网膜难以包裹炎症阑尾，易在腹腔内扩散

    4. 4.

      老年人急性阑尾炎

      • 主诉不强烈，体征不典型，临床表现轻而病理改变却很重

      • 老年人动脉硬化，易致阑尾缺血坏死

    5. 5.

      AIDS/HIV 感染病人的阑尾炎：临床症状不典型，WBC 不高

  4. 4.

    慢性阑尾炎

    • 大多由急性阑尾炎转变而来

    • 既往常有急性阑尾炎发作病史

    • 阑尾部位的局限性压痛（经常存在，位置固定）

    • 钡剂灌肠 X 线检查：阑尾变形、形态扭曲、边缘毛糙以及分节状改变，单个或多个充盈缺损等征象

  5. 5.

    阑尾肿瘤

    • 非常少见，多在阑尾切除术中或尸体解剖中被诊断

    • 阑尾类癌、阑尾腺癌、阑尾囊性肿瘤

    • 治疗方法：右半结肠切除术
