  2. 2.

    胃十二指肠溃疡的外科治疗

    1. 1.

      概述


      		 胃溃疡 十二指肠溃疡
      溃疡部位 胃小弯 球部
      发病高峰 40 ~60 岁 青壮年
      溃疡穿孔部位 胃小弯 球部前壁
      溃疡出血部位 胃小弯 球部后壁
      胃癌好发部位 胃窦
    2. 2.

      外科治疗指征：

      1. 1.

        溃疡穿孔

        1. 1.

          临床表现

          1. 1.

            病人多有溃癌病史，部分有服用阿司匹林等非甾体抗炎药病史

          2. 2.

            病人突发上腹部剧痛，呈“刀割样”，腹痛迅速波及全腹

          3. 3.

            体检见病人表情痛苦，取屈曲体位，不敢移动

          4. 4.

            全腹压痛，但以穿孔处最重 。 腹肌紧张呈“板状腹”，反跳痛明显

          5. 5.

            肠鸣音减弱或消失

          6. 6.

            叩诊肝浊音界缩小或消失

          7. 7.

            X线检查踊下可见新月状游离气体影

        2. 2.

          鉴别诊断

          1. 1.

            急性胆囊炎：右上腹绞痛或持续性疼痛伴阵发加剧，疼痛向右肩放射，Murphy征阳性

          2. 2.

            急性膜腺炎：腹痛多位于上腹部偏左并向背部放射，腹痛有一个由轻转重的过程

          3. 3.

            急性阑尾炎：体征局限于右下腹，无腹壁板样强直

        3. 3.

          外科治疗

          1. 1.

            穿孔缝合术：为主要术式，术后仍需抗溃疡药物治疗

          2. 2.

            胃大部切除术：彻底性手术，一次性解决穿孔和溃疡两个问题

      2. 2.

        溃疡大出血

        胃十二指肠溃疡大出血：因胃或十二指肠溃疡引起呕血 、大量柏油样黑便，导致红细胞计数 、血红蛋白和血细胞比容下降，病人心率加快、血压下降，甚至出现休克症状

        1. 1.

          临床表现：与出血量及速度相关

          • 呕血、黑便、血便

          • 血容量减少相关表现

          • 肠蠕动增加，肠鸣音增强

        2. 2.

          鉴别诊断

          1. 1.

            胃底食管静脉曲张破裂：有肝硬化病史，腹壁皮肤可见蜘蛛痣

          2. 2.

            应激性溃疡：有重度感染、创伤、使用激素、非甾体抗炎药等引起应激的病因，最突出临床表现：突发性上消化道大出血或穿孔

        3. 3.

          治疗

          1. 1.

            补充血容量：快速输入平衡盐溶液

          2. 2.

            放置胃管：冲洗胃腔；注入含去甲肾上腺素的生理盐水溶液

          3. 3.

            药物治疗

          4. 4.

            胃镜治疗

          5. 5.

            手术治疗指征：约10%需急症手术治疗

            • 经积极保守治疗无效者

            • 出血速度快，短期内出现休克症状者

            • 高龄病人伴有动脉硬化，出血自行停止可能

            • 经过保守治疗出血已停止，但短期内可能再次出血者

      3. 3.

        溃疡瘢痕性幽门梗阻

        1. 1.

          临床表现：

          • 主要表现为腹痛和反复呕吐

          • 呕吐物为宿食，有腐败酸臭味，不含胆汁（低氯低钾性碱中毒）

          • 脱水表现：皮肤干燥、皱缩、口唇皲裂

          • 晃动上腹部可闻“振水声”

        2. 2.

          鉴别诊断

          • 是水肿性还是瘢痕性幽门梗阻：水肿性水肿消退后通过正规的消化性溃疡药物治疗

          • 是否为胃、十二指肠降部或胰头部的肿瘤压迫所致

        3. 3.

          治疗：首选胃大部切除术

    3. 3.

      手术方式

      1. 1.

        穿孔缝合术：胃或十二指肠溃疡急性穿孔

      2. 2.

        胃大部切除术：胃十二指肠溃癌保守治疗无效或者并发穿孔、出血、幽门梗阻、癌变者

        • 毕I式：胃溃疡首选手术方式

        • 毕II式

        • Roux-en-Y术式：可防止胆胰液流入残胃招致的反流性胃炎

      3. 3.

        双侧迷走神经切断术：早期并发症--胃滞留

    4. 4.

      术后并发症

      1. 1.

        早期

        1. 1.

          术后出血

        2. 2.

          术后胃瘫：胃排空障碍

        3. 3.

          术后胃肠壁缺血坏死、吻合口破裂或漏：处理--立即禁食，放置胃管进行胃肠减压，并严密观察

        4. 4.

          十二指肠残端破裂

        5. 5.

          术后肠梗阻：多见毕II式吻合

      2. 2.

        远期

        1. 1.

          倾倒综合征（dumping syndrome）：胃大部切除术后，由于失去了幽门的节制功能，导致胃内容物排空过快，产生一系列临床症状，多见于毕II式吻合

          • 早期倾倒综合征：高渗性胃内容物快速进入肠道导致肠道内分泌细胞大量分泌血管活性物质有关

          • 晚期倾倒综合征：食物进入肠道后剌激膜岛素大量分泌，继而导致反应性低血糖

        2. 2.

          碱性性反流性胃炎：碱性肠液反流至残胃，胸骨后或上腹部烧灼痛

        3. 3.

          溃疡复发

        4. 4.

          营养性并发症：消化功能受影响，治疗应采取调节饮食，少食多餐

        5. 5.

          残胃癌：因良性疾病行胃大部切除术后5年以上，残胃出现原发癌

  3. 3.

    胃癌

    1. 1.

      病因：

      • 地域环境

      • 饮食生活因素：长期食用熏烤、盐腌食品

      • 幽门螺杆菌（HP）

      • 慢性疾病和癌前病变：胃息肉、慢性萎缩性胃炎、残胃

      • 遗传和基因

    2. 2.

      病理

      1. 1.

        大体类型

        1. 1.

          早期胃癌：仅限于黏膜或黏膜下层

          • 小胃癌：直径在10mm以下

          • 微小胃癌：直径在5mm以下

        2. 2.

          进展期胃癌：超过黏膜下层

      2. 2.

        扩散与转移

        • 直接浸润

        • 淋巴转移:是胃癌的主要转移途径，通常是循序渐进的，先向胃周淋巴结转移

        • 血行转移：常见转移的器官有肝（最多见）、肺、胰、骨髓等

        • 腹膜种植转移

          • 直肠前凹转移癌

          • Krukenberg瘤：女性病人胃癌形成卵巢转移性肿瘤

    3. 3.

      临床表现

      • 锁骨上淋巴结肿大( Virchow' s sentinel node)

      • 贲门胃底癌--胸骨后疼痛和进食梗阻感

      • 幽门附近胃癌--呕吐物多为隔夜宿食和胃液

      • 肿瘤破溃--呕血、黑便

    4. 4.

      诊断：早期诊断是提高治愈率的关键

      • 电子胃镜检查：是诊断胃癌的最有效方法；直接观察胃黏膜病变的部位和范围，钳取小块组织作病理学检查

      • X线坝餐检查：优点是痛苦小；X线征象主要有龛影、充盈缺损

      • CT检查：手术前判断肿瘤N分期和M分期的首选方法

      • 正电子发射成像技术( PET)：利用胃癌组织对（FDG）的亲和性，判断淋巴结和远处转移病灶情况

    5. 5.

      治疗：治疗策略是以外科手术为主的综合治疗

      1. 1.

        旱期胃癌的内镜下治疗：直径小于2cm的无溃疡表现的分化型黏膜内癌

      2. 2.

        手术治疗

        • 根治性手术

          • 标准术式是：D2淋巴结清扫的胃切除术

        • 姑息性手术

      3. 3.

        化学治疗：根治性手术后辅助化疗的目的是控制残存的肿瘤细胞以减少复发的机会，进展期胃癌根治术后 无论有无淋巴结转移均需化疗
