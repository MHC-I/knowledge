38.肝疾病

  1. 1.

    解剖生理概要

    1. 1.

      肝是人体内最大的实质性脏器，有两个面

    2. 2.

      肝圆韧带：脐动脉退化而来，可用做胆道修补

    3. 3.

      肝蒂：肝十二指肠韧带包含门静脉、肝动脉 、淋巴管 、淋巴结和神经

    4. 4.

      肝门

      肝门 描述 方向
      第一肝门 门静脉 、肝动脉和肝总管在肝脏面的横沟处各自分出左、右干进入肝实质 入肝
      第二肝门
      		 三条主要的肝静脉在肝后上方的静脉窝进入下腔静脉 出肝
      第三肝门 小部分肝血液经数支肝短静脉汇入肝后方的下腔静脉 出肝
    5. 5.

      肝的分叶和分段（5 叶 8 段）

      分叶 分段
      尾状叶 Ⅰ 段：尾状叶
      左外叶 Ⅱ 段：左外叶上段
      Ⅲ 段：左外叶下段
      左内叶 Ⅳ 段：左内叶
      右前叶 Ⅴ 段：右前叶下段
      Ⅷ 段：右前叶上段
      右后叶 Ⅵ 段：右后叶下段
      Ⅶ 段：右后叶上段
    6. 6.

      肝的血液供应：肝 70～75% 的血液由门静脉供应，40～60% 氧由肝动脉供应

    7. 7.

      肝的储备功能和再生能力均很强大，故可行半肝乃至更大范围切除术；但对缺氧敏感，常温下肝门阻断的时间不宜超过 15 ~20 分钟

    8. 8.

      肝的生理功能

      • 分泌胆汁：帮助脂肪消化及脂溶性维生素的吸收

      • 代谢功能：营养物质经门静脉系统进入肝

      • 凝血功能

      • 解毒作用

      • 吞噬或免疫作用

  2. 2.

    原发性肝恶性肿瘤

    1. 1.

      肝细胞癌（肝癌）（HCC）

      1. 1.

        病因：肝硬化、病毒性肝炎、黄曲霉素、环境因素

      2. 2.

        病理

        • 病理组织：肝细胞型、胆管细胞型、混合型（二者同时出现）

        类型 直径
        微小肝癌 ≤2cm
        小肝癌 2~5
        大肝癌 5~10
        巨大肝癌 >10

      3. 3.

        播散途径

        • 肝癌细胞极易经门静脉系统在肝内播散--最常见（🥦 模式）

        • 血行肝外转移最多见于肺

        • 中晚期病例，肿瘤可直接侵犯邻近脏器及横隔（转移到肺）

      4. 4.

        临床表现

        • 易发年龄：40～50 岁，男 > 女

        • 一旦出现症状和体征，疾病多已进入中、晚期

        • 可能有肝区疼痛、肝大或右上腹肿块；乏力、消瘦、食欲减退、黄疸、腹胀

        • 并发症：① 肝性昏迷；② 上消化道出血；③ 癌肿破裂出血； ④ 继发感染

      5. 5.

        诊断：

        • 病人有乙或丙型肝炎等肝病病史

        • 甲胎蛋白(AFP) ≥400ng/ml（妊娠、活动性肝病、生殖腺胎胚源性肿瘤等 AFP 亦可升高）

        • 超声、CT 或 MRI 检查发现肝实质性肿块，且具有肝细胞癌典型影像学表现

      6. 6.

        治疗

        • 原则：早期诊断、早期采用以手术切除为主的综合治疗

        1. 1.

          部分肝切除：首选和最有效方法

          1. 1.

            安全性评估

            1. 1.

              一般情况

              • 较好 ，无明显心 、肺 、肾等重要脏器器质性病变

              • Child-Pugh 肝功能分级属 A 级（或短期治疗后可恢复到 A 级）

              • 有条件的医院，术前可做 ICG 检测

              • 评估肝切除后残肝体积，手术后足够维持肝功能

            2. 2.

              可切除性评估

              • 没有肝外多处转移

              • 单发的微小肝癌和小肝癌

              • 单发的向肝外生长的 肝癌或巨大肝癌，受肿瘤破坏的肝组织少于 30%，肿瘤包膜完整，周围界限清楚

              • 多发肿瘤，但肿瘤结节少于 3 个，且局限在肝的一段或一叶内

        2. 2.

          肝移植：原则上选择肝功能 C 级的小肝癌病例行肝移植

        3. 3.

          对不能切除的肝癌的外科治疗：肝动脉结扎、肝动脉化疗栓塞（TACE）、射频、冷冻、激光、微波等

        4. 4.

          术后复发肝癌的再手术治疗：

          • 肝癌即使获得根治性切除，5 年内仍有 60%～70% 的病人出现转移、复发

          • 一般情况良好、肝功能正常，病灶局限允许切除，可施行再次切除

        5. 5.

          肝癌破裂出血：行肝动脉结扎或动脉栓塞术

    2. 2.

      肝内胆管癌（ICC）

      1. 1.

        多源于肝内胆管上皮细胞，多为腺癌

      2. 2.

        临床表现：多与 HCC 相似，最常见的症状是右上腹疼痛和体重减轻

      3. 3.

        CT 和 MRI：表现为局灶性肝肿块，肿块周围的胆管可能扩张

      4. 4.

        治疗有效方法是肝切除，放化疗效果有限

  3. 3.

    转移性肝肿瘤

    1. 1.

      原发肿瘤主要为：结直肠癌、胃癌、胰腺癌

    2. 2.

      临床表现：

      • 早期无症状，随瘤体增大上腹或肝区不适或隐痛

      • 病情加重时，可出现乏力、发热、体重下降

      • 晚期可出现贫血、黄疸、腹水

      • 体检发现肝大，有时可触及坚硬的癌结节

    3. 3.

      治疗

      • 单发的转移性肝肿瘤，最有效的治疗方法是肝切除

      • 手术原则：完全切除肿瘤（切缘距肿瘤 >1cm）最大限度保留健康肝组织

  4. 4.

    肝良性肿瘤

    1. 1.

      肝海绵状血管瘤最常见

    2. 2.

      临床特点

      • 常见于中年女性

      • 生长缓慢（数年）

      • 瘤体较小时无任何临床症状，增大后主要表现为肝大或压迫胃、十二指肠等邻近器官，引起上腹部不适、腹胀、暧气、腹痛等症状

      • 体格检查：腹部肿块与肝相连，表面光滑，质地柔软，有囊性感及不同程度的压缩感

    3. 3.

      治疗：

      • 手术切除是治疗肝海绵状血管瘤的最有效的方法（小的 、无症状的无需治疗，只需注意每 6～12 个月超声检查）

      • 有明显症状，或肿瘤直径 >10cm，特别是位于肝缘者，可行手术切除

      • 最危险的并发症是肿瘤破裂引起的大出血

  5. 5.

    细菌性肝脓肿

    1. 1.

      细菌侵入肝途径

      • 胆道：主要途径，胆道梗阻并发生化脓性胆管炎时，细菌沿着胆管上行

      • 门静脉：坏疽性阑尾炎可突破肠道屏障入肝

      • 肝动脉：体内任何部位的化脓性病变可经该途径入肝

      • 淋巴系统

      • 直接经伤口侵入

    2. 2.

      致病菌：肺炎克雷伯菌、大肠埃希菌、厌氧链球菌、葡萄球菌

    3. 3.

      病史：继发于胆道感染或其他化脓性疾病，多有糖尿病病史

    4. 4.

      临床表现

      • 典型症状是寒战、高热、肝区疼痛和肝大

      • 体温可达 39～40°C，伴恶心、呕吐、食欲缺乏和周身乏力

      • 肝区钝痛或胀痛多属持续性，可伴右肩牵涉痛，右下胸及肝区叩击痛，肿大的肝有压痛

      • 严重时或并发胆道梗阻者，可出现黄疸

      • 肝右叶脓肿可穿破肝包膜形成膈下脓肿，或突入右侧胸腔；向腹腔穿破，则发生急性腹膜炎

    5. 5.

      实验室检查

      • 白细胞计数 ↑ （中性粒性细胞为主）

      • 转氨酶和碱性磷酸酶 ↑

      • CRP 增高，ESR 延长

    6. 6.

      影像学检查

      • 超声：可明确其部位和大小，为首选检查方法

      • CT：更易显示多发小脓肿

      • MRI：对存在可疑胆道疾病时帮助较大

      • X 线胸腹部检查 ：右叶脓肿可使右膈肌升高，肝阴影增大或有局限性隆起，有时出现右侧反应性胸膜炎或胸腔积液

    7. 7.

      诊断：必要时可在肝区压痛最剧处或超声引导下施行诊断性穿刺予以确诊

    8. 8.

      治疗

      1. 1.

        全身支持治疗：营养支持、增强机体抵抗力

      2. 2.

        抗生素治疗：经验用药，三代头抱联合应用甲硝唑

      3. 3.

        经皮肝穿刺脓肿置管引流术

      4. 4.

        手术治疗

        • 手术方式：切开引流

        • 适应征

          • 脓肿较大、分隔较多

          • 己穿破胸腔或腹腔

          • 胆源性肝脏肿，慢性肝脓肿

        • 注意事项

          • 脓肿已向胸腔穿破者，应同时引流胸腔

          • 胆道感染引起的肝脏肿，应同时引流胆道

          • 血源性肝脏肿，应积极治疗原发感染灶

          • 慢性肝脏肿，一般需施行肝切除治疗

  6. 6.

    肝棘球蚴病

    1. 1.

      病因病理：

      • 肝包虫病是临床上最常见的一种棘球蚴病，主要流行于西部畜牧地区和半农半牧区，以细粒棘球病最多见；终宿主以犬最常见，中间宿主以羊最常见，人与人之间不传染

      • 侵入体内的六钩蚴在肝内先发育成小的囊体，囊体长大并挤压肝实质，在肝内形成一个具有多层 壁结构和多种内容物的囊性肿块(肝包虫囊肿)。随着囊肿的膨胀性生长，周围肝实质受压，肝细胞变性、萎缩、消失，囊肿周围的管道系统纤维化，在外囊与肝实质之间形成一层纤维膜状结构。

    2. 2.

      临床表现

      1. 1.

        包虫囊破裂

        • 囊内容溢入腹腔，可导致严重过敏反应；子囊种植产生多发囊肿，出现腹胀或导致肠梗阻

        • 囊内容破溃入胆道，可引起梗阻性黄疸或反复发作的胆管炎

        • 经横膈，破裂入胸腔（甚至肺），导致反复肺部感染，可能咳出子囊

      2. 2.

        包虫囊肿压迫

        • 压迫胆管出现黄疸

        • 压迫肝静脉引起巴德-吉亚利综合征

      3. 3.

        感染：多由胆瘘引起继发性细菌感染

      4. 4.

        过敏反应：虫体抗原进入血液循环引起寻麻疹

      5. 5.

        虫体抗原沉积肾小球引起膜性肾小球肾炎

    3. 3.

      诊断

      • 流行地区居住史，犬、羊接触史

      • 超声检查：诊断准确率高，是筛选和初步诊断的首选方法，可帮助确定包虫的发育阶段和分型

      • X 线检查：外囊钙化时，可显示环形或弧形钙化影

      • CT 和 MRI 检查：能显示囊肿与肝内结构的解剖关系（疑有胆道受累时，可行 MRCP 检查）

    4. 4.

      治疗

      1. 1.

        手术治疗

        • 原则：尽量完整摘除外囊，清除内囊，避免囊液外溢，防止复发；合理处理残腔及胆瘘，减少术后并发症

        1. 1.

          外囊完整剥（切）除术：较好解决了术后复发和残腔并发症的问题，可作为根治性手术的首选方式

        2. 2.

          内囊摘除术：是经典的手术方式，关键是避免囊液外溢和头节的灭活

      2. 2.

        药物治疗：阿苯达唑

      3. 3.

        超声引导下经皮肝穿刺抽吸术：适用于体积较小、位于肝组织内的 I 型囊肿；不适用于囊肿和胆管相通的病人
