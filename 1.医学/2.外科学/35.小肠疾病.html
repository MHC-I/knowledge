35.小肠疾病

  1. 1.

    肠梗阻

    1. 1.

      概述

      1. 1.

        肠梗阻（intestinal obstruction）：任何原因引起的肠内容物通过障碍统称肠梗阻，是常见的外科急腹之一

      2. 2.

        病因和分类（在病理过程中可相互转化）

        1. 1.

          按梗阻原因分

          • 机械性：肠外因素(粘连性最常见)、肠壁因素(如肠套叠、肠扭转)、肠腔内因素(如蛔虫梗阻)

          • 动力性（易忽视）

            • 麻痹性：低钾血症

            • 痉挛性（最常见）

          • 血运性（最严重）

        2. 2.

          按肠壁血运有无障碍分


          		 单纯性肠梗阻 绞窄性肠梗阻
          腹痛 阵发性 ，轻度中度 持续性，重度
          呕吐 较少 严重
          休克 少见 出现早，常见
          发热 不明显 明显
          腹痛和肌卫 不明显 出现早，明显
          补液治疗 效果好 效果差
        3. 3.

          按梗阻部位分

          • 高位（空肠）梗阻

          • 低位小肠（回肠）和结肠梗阻--又称“闭袢性梗阻”

        4. 4.

          按梗阻程度分

          • 完全性

          • 不完全性

      3. 3.

        病理和病理生理

        1. 1.

          局部变化：

          • 梗阻以上肠蠕动增加，肠腔膨胀；梗阻以下肠管则瘪陷；扩张肠管和塌陷肠管交界处即为梗阻所在

          • 肠腔压力不断升高，可使肠壁静脉回流受阻，肠壁充血水肿，液体外渗；肠壁及毛细血管通透性增加，肠壁上有出血点

        2. 2.

          全身变化

          1. 1.

            水、电解质和酸碱失衡

            • 体液在第三间隙丢失：胃肠道分泌的液体不能被吸收返回全身循环而积存在肠腔，同时肠壁继续有液体向肠腔内渗出

            • 高位肠梗阻代谢性碱中毒：呕吐

            • 低位肠梗阻代谢性酸中毒：碱性消化液不能返回全身、组织灌注不良

          2. 2.

            血容量下降

          3. 3.

            休克

          4. 4.

            呼吸和心脏功能障碍：横膈上升，影响呼吸；腹内压升高，下腔静脉回流减少

      4. 4.

        临床表现

        1. 1.

          症状：

          1. 1.

            腹痛：

            • 阵发性绞痛：梗阻部位以上强烈肠蠕动，发生腹痛 ➡️ 肠管肌过度疲劳而暂时性弛缓

            • 腹痛时伴高亢肠鸣音；有积气积液时，呈气过水声或高调金属音

            • 腹痛间歇期缩短至持续腹痛应警惕绞窄性肠梗阻

            • 麻痹性肠梗阻（瘫痪）无腹痛

          2. 2.

            呕吐

            • 高位梗阻：呕吐出现早，频繁，吐出物为胃、十二指肠内容物

            • 低位梗阻：呕吐出现晚，次数少，吐出物初为胃内容物，后为经发酵、腐败呈粪样的肠内容物

            • 麻痹性肠梗阻：呕吐为溢出性（麻痹）

          3. 3.

            腹胀：高位不明显，可见胃型；低位及麻痹性肠梗阻明显，可见肠型

          4. 4.

            排气排便停止：初期可有残余气便排出

        2. 2.

          体征

          1. 1.

            视：肠型和蠕动波(机械性)

          2. 2.

            触：轻微压痛但无腹膜刺激征(单纯性);固定压痛且有腹膜刺激征(绞窄性)

          3. 3.

            叩：移动性浊音阳性(绞窄性)

          4. 4.

            听：肠鸣音亢进(机械性);肠鸣音减弱/消失(麻痹性)

        3. 3.

          辅助检查

          1. 1.

            化验检查：由于失水和血液浓缩，白细胞计数、血红蛋白和血细胞比容都可增高，尿比重也增高

          2. 2.

            X 线检查（最主要）

            • 立位片可见多数液平和气胀肠袢，卧位片可见肠形和扩张程度

            • 特征

              • 鱼肋征（空肠梗阻）

              • 阶梯状液平（回肠梗阻）

              • 结肠袋形（结肠梗阻）

      5. 5.

        诊断

        1. 1.

          是否肠梗阻：四大症状

        2. 2.

          是机械性还是动力性梗阻：X 和 CT 检查最有价值

          • 麻痹性肠梗阻显示大、小肠全部充气扩张

          • 机械性肠梗阻胀气限于梗阻以上的部分肠管

        3. 3.

          是单纯性还是绞窄性梗阻（极其重要）：下列需考虑绞窄性

          1. 1.

            持续性剧烈腹痛/伴阵发性加剧

          2. 2.

            早期出现休克且抗休克无效

          3. 3.

            有腹膜炎表现

          4. 4.

            腹胀不对称，有局部隆起/压痛肿块

          5. 5.

            呕吐出现早且频发、血性

          6. 6.

            X 线检查见孤立扩大的肠袢

          7. 7.

            经积极的非手术治疗无明显改善

        4. 4.

          是高位还是低位梗阻：呕吐特点、X 线特点

        5. 5.

          是完全性还是不完全性梗阻：不完全性梗阻症状轻、体征不明显

        6. 6.

          是什么原因引起梗阻：

          • 粘连性：最常见，多有手术、炎症史

          • 先天畸形：多见于新生儿

          • 肠套叠：多见于 2 岁以内小儿

          • 蛔虫：多见于儿童

          • 肿瘤/粪块堵塞：多见于老年人

      6. 6.

        治疗：

        1. 1.

          非手术治疗

          1. 1.

            胃肠减压

          2. 2.

            纠正水、电解质紊乱和酸碱失衡

          3. 3.

            防治感染

          4. 4.

            吸氧、抑制胃液分泌

        2. 2.

          手术治疗

          1. 1.

            适应证：绞窄性肠梗阻、肿瘤/先天畸形所致肠梗阻、非手术治疗无效

          2. 2.

            出现以下情况表明肠管已无生机

            • 肠壁已呈紫黑色并已塌陷

            • 肠壁己失去张力和蠕动能力，对刺激无收缩反应

            • 相应的肠系膜终末小动脉无搏动

            • 必要时采用盐水纱布热敷或在肠系膜血管根部注射 1% 普鲁卡因/酚妥拉明后观察 15～30min 再判断，仍不能判断者可将其回纳、关腹，18～24h 后再次剖腹探察

    2. 2.

      粘连性肠梗阻

      1. 1.

        最常见(40～60%)的一种肠梗阻

      2. 2.

        病因：先天性（发育异常、胎粪性腹膜炎）、后天性（有腹部手术史/感染史）--最多见

      3. 3.

        病理：多发生在小肠

      4. 4.

        诊断：机械性肠梗阻表现，腹腔手术、感染病史

        • 慢性肠梗阻症状或多次急性发作者多为广泛粘连引起的梗阻

        • 突然出现急性梗阻症状，出现腹膜剌激征，应考虑绞窄性肠梗阻

      5. 5.

        治疗：争取非手术治疗，出现绞窄性时尽快手术治疗

      6. 6.

        预防：

        1. 1.

          清除手套上的淀粉、滑石粉，不遗留异物于腹腔内，减少肉芽组织的产生

        2. 2.

          减少缺血组织，不作大块组织结扎

        3. 3.

          注意无菌操作，减少炎性渗出

        4. 4.

          保护肠浆膜面，防止损伤与干燥

        5. 5.

          冲洗清除腹腔内积血、积液，必要时放置引流

        6. 6.

          及时治疗腹腔内炎性病变，防止炎症扩散

        7. 7.

          术后早期活动和促进肠蠕动及早恢复

    3. 3.

      肠扭转

      1. 1.

        肠扭转（volvulus）：是一段肠袢及其系膜沿其系膜长轴扭转 360°～720° 而造成的闭袢型肠梗阻。 闭袢性肠梗阻 + 绞窄性肠梗阻(病情凶险而发展迅速)

      2. 2.

        临床表现

        急性小肠扭转 乙状结肠扭转
        好发 青壮年 青年(急性)
        老年男性(亚急性)
        病史 饱餐后剧烈活动 习惯性便秘
        表现 腹部持续性绞痛阵发性加剧、可放射到腰背部
        频繁呕吐（高位）
        腹胀不明显
        		 腹部持续胀痛
        呕吐不明显（低位）
        明显进行性腹胀
        检查 腹部平片：绞窄性肠梗阻、空回肠换位、多形态小跨度卷曲肠襻 钡剂灌肠：扭转部位鸟嘴征
        腹平片：马蹄状巨大双腔充气肠袢
        结肠镜:确诊且可依此复位
      3. 3.

        治疗：及时手术治疗

    4. 4.

      肠套叠

      1. 1.

        肠套叠（intestinal intussusception）：肠的一段套入其相连的肠管腔内

      2. 2.

        临床表现：一般多见于幼儿，而慢性复发性肠套叠，多见于成人

        • 腹痛：突然发作剧烈的阵发性腹痛

        • 血便：果酱样血便

        • 腹部肿块：常于脐右上方扪及腊肠形、表面光滑、稍可活动、具有压痛的肿块

      3. 3.

        治疗

        • 钡剂灌肠：不仅是诊断方法，也是一种有效的治疗方法；钡剂受阻，示杯口状阴影(结肠套叠)、弹簧状阴影(小肠套叠)

        • 手术治疗：适用于灌肠治疗无效、病程 > 48h、疑有肠坏死、出现腹膜刺激征

  2. 2.

    肠感染性疾病

    1. 1.

      肠结核

    2. 2.

      肠伤寒

      • 特点：穿孔后反有脉搏增快、白细胞计数增加、体温下降

      • 治疗：确诊后及时手术治疗，原则是施行穿孔缝合术，手术应简单、快速，必要时才考虑做肠切除

  3. 3.

    肠炎性疾病

    1. 1.

      急性出血性肠炎

      1. 1.

        病因：C 型魏氏杆菌产生 β 毒素；肠道内缺乏足够破坏 β 毒素的膜蛋白酶

      2. 2.

        病理：多位于空回肠(特别是空肠)、可呈节段性；肠管扩张、肠壁各层充血水肿、广泛出血、坏死、溃疡形成，甚至穿孔

      3. 3.

        临床表现：

        • 急性腹痛：阵发性绞痛或持续性疼痛伴阵发性加剧

        • 腹胀、呕吐、腹泻

        • 血水样便或果酱样腥臭便

      4. 4.

        治疗：一般非手术治疗；手术指征：

        • 有明显腹膜炎表现，或腹腔穿刺有脓性或血性渗液，怀疑有肠坏死或穿孔

        • 不能控制的肠道大出血

        • 有肠梗阻表现经非手术治疗不能缓解

    2. 2.

      克罗恩病

  4. 4.

    肠系膜血管缺血性疾病

    1. 1.

      病因：

      • 肠系膜上动脉栓塞（血栓多来自心脏）

      • 肠系膜上动脉血栓形成（发生于动脉硬化性阻塞或狭窄）

      • 肠系膜上静脉血栓形成（血液淤滞、高凝）

    2. 2.

      病理：绞窄性动力性肠梗阻

    3. 3.

      临床表现：

      • 发病急骤，多伴频繁（血性）呕吐，早期症状明显且严重，特点是严重的症状与轻微的体征不相称

      • X 线平片早期显示受累小肠 、结肠扩张胀气；晚期显示腹部普遍密度增高

      • 选择性动脉造影有助于诊断

    4. 4.

      治疗：及早诊断，及早治疗，包括支持疗法和手术治疗

  5. 5.

    短肠综合征

    1. 1.

      短肠综合征（short bowel syndrome ,SBS）：是指小肠被广泛切除后，残存的功能性肠管不能维持病人营养需要的吸收不良综合征

    2. 2.

      病理生理：残存小肠少于 75cm(有完整结肠)，或丧失回盲瓣、残存小肠少于 100cm 者导致短肠综合征

    3. 3.

      临床表现：腹泻（最早出现）、水和电解质失衡，以及营养不良

    4. 4.

      治疗：

      • 首在预防，在处理小肠疾病时，应尽量避免不必要的扩大切除

      • 营养支持是主要治疗方法，手术治疗（减少肠道运行，增加肠表面积）
