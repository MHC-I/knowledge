22.颈部疾病

一、单纯甲状腺肿

  1. 1.

    病因

    1. 1.

      甲状腺素原料（碘）缺乏：甲状腺素⬇️-→垂体TSH ⬆️-→刺激甲状腺增生和代偿性肿大

    2. 2.

      甲状腺素需要量增高：青春期、妊娠期或绝经期妇女(生理性甲状腺肿)

    3. 3.

      甲状腺合成分泌障碍

  2. 2.

    临床表现

    1. 1.

      女性多见

    2. 2.

      腺体表面光滑，质地柔软，随吞咽上下移动

    3. 3.

      肿大结节压迫周围器官引起症状

  3. 3.

    预防：加碘食盐

  4. 4.

    治疗

    1. 1.

      生理性甲状腺肿： 多食含腆丰富的海带、紫菜

    2. 2.

      20岁以下病人：小量甲状腺素或优甲乐

    3. 3.

      手术治疗：甲状腺次全切除术

      指征：

      1. 1.

        因气管、食管或喉返神经受压引起临床症状者

      2. 2.

        胸骨后甲状腺肿

      3. 3.

        巨大甲状腺肿影响生活和工作者

      4. 4.

        结节性甲状腺肿继发功能亢进者

      5. 5.

        结节性甲状腺肿疑有恶变者

二、甲亢

甲状腺功能亢进(hyperthyroidism)：是由各种原因引起循环中甲状腺素异常增多而出现以全身代谢亢进为主要特征的疾病总称

原发性最多见：两侧对称肿大，常伴有眼球突出

  1. 1.

    临床表现

    1. 1.

      性情急躁、容易激动、多汗

    2. 2.

      食欲亢进但却消瘦、体重减轻

    3. 3.

      心悸、脉快有力(100次/分以上)、脉压增大

    4. 4.

      内分泌紊乱

  2. 2.

    诊断

    1. 1.

      基础代谢率测定

      • 基础代谢率=(脉率+脉压)-111

        • 正常值为±10%

        • +20% ~30%为轻度

        • +30% ~60%为中度

        • +60%以上为重度

    2. 2.

      甲状腺摄I131测定

    3. 3.

      血清中T3和T4的测定

      • T3高于正常4倍

      • T4高于正常2.5倍

  3. 3.

    手术治疗

    1. 1.

      指征

      1. 1.

        继发性甲亢或高功能腺瘤

      2. 2.

        中度以上的原发性甲亢

      3. 3.

        伴压迫症状或胸骨后甲肿

      4. 4.

        治疗复发或长期用药有困难者

      5. 5.

        妊娠早 、中期的甲亢病人凡具有上述指征者，应考虑手术治疗，并可以不终止妊娠

    2. 2.

      禁忌

      1. 1.

        青少年病人

      2. 2.

        症状较轻者

      3. 3.

        不能耐受手术者(老人、器质性疾病)

    3. 3.

      药物准备

      1. 1.

        抗甲状腺药物加腆剂：服用硫脲类药物后加用腆剂 2 周

      2. 2.

        单用腆剂：

        • 适合症状不重，以及继发性甲亢和高功能腺瘤病人

        • 脉率<90次/分，基础代谢率<20% ，便可进行手术

      3. 3.

        普茶洛尔:对于a、b不能耐受或无效者

    4. 4.

      术后并发症

      1. 1.

        术后呼吸困难：最严重，多发生在术后48小时内

        • 出血及压迫气管

        • 喉头水肿

        • 气管塌陷

        • 双侧喉返神经损伤

      2. 2.

        喉返神经损伤

        • 一侧-→声嘶

        • 双侧-→失音或严重的呼吸困难

      3. 3.

        喉上神经

        • 外支-→环甲肌瘫痪，声带松弛、音调降低

        • 内支-→喉部黏膜感觉丧失，容易误咽发生呛咳

      4. 4.

        甲状旁腺功能减退

        • 误伤甲状旁腺或其血液供给受累所致

        • 抽搐发作时， 立即静脉注射10%葡萄糖酸钙或氯化钙10~20ml

      5. 5.

        甲状腺危象:是甲亢的严重并发症，是因甲状腺素过量释放引起的暴发性肾上腺素能兴奋现象

        • 主要表现:高热(>39°C)、脉快(>120次/分)

        • 预防最关键措施：术前使基础代谢率降至正常水平下

        • 治疗：

          • 一般治疗

          • 碘剂

          • 肾上腺素能阻滞剂

          • 氢化可的松

三、甲状旁腺功能亢进

  1. 1.

    生理

    • 甲状旁腺分泌甲状旁腺素(PTH)，其主要靶器官为骨和肾。 PTH 的生理功能是调节体内钙的代谢并维持钙和磷的平衡。

    • 甲状旁腺功能亢进时，可出现高血钙、高尿钙和低血磷。

    • PTH不受垂体控制而与血钙离子浓度之间存在反馈关系

  2. 2.

    临床表现

    • 无症状型病例可仅有骨质疏松等非特异性症状

    • I型(骨)：骨膜下骨质吸收，最常见于中指桡侧或锁骨外 1/3 处

    • II型(肾)：以肾结石为主

    • III型：I+II

  3. 3.

    诊断

    • 血钙测定：是发现甲状旁腺功能亢进的首要指标

    • 血磷测定

    • PTH测定：PTH升高是诊断甲状旁腺功能亢进最可靠的直接证据

    • 尿中环腺苷酸( cAMP)的测定

  4. 4.

    治疗

    • 甲状旁腺腺瘤：切除

    • 甲状旁腺增生

      ①作甲状旁腺次全切除

      ②切除所有4枚甲状旁腺，同时作甲状旁腺自体移植

    • 甲状旁腺癌：作整块切除，且应包括一定范围的周围正常组织

四、甲状腺炎

亚急性

  • 常继发于病毒性上呼吸道感染

  • 临床表现：甲状腺突然肿胀、发硬、吞咽困难及疼痛， 并向病侧耳颞处放射

  • 诊断：血清 T3、T4 浓度升高，但甲状腺摄取I-131量显著降低(分离现象)

  • 治疗：泼尼松

慢性淋巴细胞性(桥本)

  • 自身免疫性疾病，甲减最常见原因

  • 临床表现：无痛、对称、质硬、表面光滑

  • 诊断：基础代谢率低

  • 治疗：甲状腺素、优甲乐

五、甲状腺腺瘤

  • 滤泡状腺瘤多见，多发于40岁以下的妇女

  • 临床表现：多单发，稍硬，表面光滑，无压痛

  • 腺瘤包膜完整，结节性甲状腺肿包膜不完整

六、甲状腺癌

  1. 1.

    病理

    • 分化癌

      • 乳头状癌：成人的最主要类型，儿童的全部，恶性程度低，多见于30 ~45 岁女性

      • 滤泡状腺癌：见于 50 岁左右中年人

      • 髓样癌：来源于滤泡旁降钙素分泌细胞 (C 细胞)

    • 未分化癌：高度恶性，见于 70 岁左右老年人

  2. 2.

    临床表现：

    • 甲状腺内发现肿块

    • 当肿瘤侵犯喉返神经可出现声音嘶哑;交感神经受压引起Horner综合征

  3. 3.

    诊断：

    • 存在多年的甲状腺肿块，在短期内迅速增大

    • 超声

    • 细针穿刺细胞学检查

    • 血清降钙素测定

  4. 4.

    治疗：

    1. 1.

      手术治疗

      1. 1.

        甲状腺全切或近全切(满足任意一条)：

        • 与腺叶切除术相反

          • 颈部有放射史

          • 已有远处转移

          • 甲状腺外侵犯

          • 有不良病理类型

        • 肿块直径大于 4cm

        • 双侧癌结节

      2. 2.

        腺叶切除(全部满足)：

        • 无颈部有放射史

        • 无远处转移

        • 无甲状腺外侵犯

        • 无其他不良病理类型

        • 肿块直径小于 1cm

    2. 2.

      放射性核素治疗：I-131

    3. 3.

      TSH 抑制治：终身服用甲状腺素片或左甲状腺素

    4. 4.

      放射外照射治疗：主要用于未分化型甲状腺癌

七、甲状腺结节

  • 病史：若过去存在甲状腺结节，近日突然快速、无痛增大，应考虑癌肿，男性更应重视

  • 体格检查：明显的孤立结节是最重要的体征，癌肿病人常于颈部下 1/3 处触及大而硬的淋巴结

  • 血清学检查：TSH 水平与甲状腺结节的良恶性相关；降钙素水平> l00pg/ml 提示髓样癌

  • 超声检查：敏感性高，提供结节供血情况，有助于良恶性的鉴别

  • 核素显像：功能越低下，血供越丰富，结节为恶性的几率越大

  • 针眼涂片细胞学检查：确定结节性质最可靠的方法

八、颈淋巴结结核

  • 多见于儿童和青年人

  • 常为结核杆菌经扁桃体、龋齿侵入所致

  • 淋巴结较硬，无痛，可推动-→不易推动的结节性肿块-→寒性脓肿-→慢性溃疡
