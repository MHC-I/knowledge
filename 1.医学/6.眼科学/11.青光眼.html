11.青光眼

  1. 1.

    概述

    1. 1.

      青光眼（glaucoma）：一组以特征性视神经萎缩和视野缺损为共同特征的疾病，病理性眼压增高为主要危险因素。（第二致盲疾病，且不可治疗）

    2. 2.

      眼压与青光眼

      • 眼压：眼内容物作用于眼球内壁的压力

      • 正常眼压：10～21mmHg；双眼的差值 <５mmHg；一天内眼压的变化 <８mmHg

      • 眼压是唯一证实的视神经危险因素：越高，持续时间越长，视神经损害的风险越大

      • 高眼压并非都是青光眼，而正常眼压也不能排除青光眼

        • 高眼压症：部分患者眼压虽已超越统计学正常上限，但长期随访并不出现视神经、视野损害

        • 正常眼压青光眼（NTG）：部分患者眼压在统计学正常范围内，却发生了典型青光眼视神经萎缩和视野缺损

    3. 3.

      房水循环途径（Aqueous humor’s pathway）

      • 房水自睫状突生成后，经后房越过瞳孔到达前房，然后主要通过两个途径外流：

        • ① 小梁网通道：经前房角小梁网进入 Schlemm 管，再通过巩膜内集合管至巩膜表层睫状前静脉;

        • ② 葡萄膜巩膜通道：通过前房角睫状体带进入睫状肌间隙，然后进入睫状体和脉络膜上腔，最后穿过巩膜胶原间隙和神经血管间隙出眼

      • 眼压高低主要取决于房水循环中的 3 个因素：

        • 睫状突生成房水的速率

        • 房水通过小梁网流出的阻力

          • 房水引流系统异常（开角型青光眼）

          • 周边虹膜堵塞了房水引流系统（闭角型青光眼）

        • 上巩膜静脉压

    4. 4.

      临床诊断标准

      • 眼压：Goldmann 眼压计是眼压测量的金标准

      • 房角：诊断开角/闭角型青光眼的依据，最好方法是通过房角镜检查直接观察房角结构

      • 视野：视野改变是诊断青光眼的金标准

      • 视盘：视杯扩大是青光眼视盘损害的重要特征

  2. 2.

    原发性青光眼

    1. 1.

      闭角型青光眼（PACG）

      1. 1.

        PACG：由于周边虹膜堵塞小梁网，或与小梁网产生永久性粘连，房水外流受阻，引起眼压升高造成视神经和视野损害的一类青光眼。患眼具有房角狭窄，周边虹膜易于与小梁网接触的解剖特征。

      2. 2.

        房角关闭机制

        • 瞳孔阻滞型

        • 非瞳孔阻滞型

          • 周边虹膜肥厚型

          • 睫状体前位型

        • 多种机制共存型

      3. 3.

        原发性闭角型青光眼的激发试验

        1. 1.

          暗室试验：让受试者在暗室中静坐 1 小时后眼压升高 ≥8mmHg 为阳性

        2. 2.

          暗室俯卧试验：患者俯卧时要求背部平衡，眼球不能受压，1.5 小时后测眼压，如果眼压较俯卧前增高 ≥8mmHg 以上为阳性

      4. 4.

        急性（acute ACG）

        1. 1.

          急性闭角型青光眼(acute angle-closure glaucoma)：是一种以房角突然关闭，导致眼压急剧升高并伴有相应症状和眼前段组织病理改变为特征的眼病。

        2. 2.

          特点：

          • 多见于 50 岁以上老年人，女性更常见

          • 绝大部分双眼先后或同时发病

          • 起病急，可严重损害视功能，短期内可导致失明

        3. 3.

          病因：瞳孔阻滞是发病的最重要因素

          • 诱发因素：情绪激动、暗室停留时间过长、抗胆碱药物的应用、长时间阅读 、疲劳及疼痛

          • 解剖因素(具有遗传倾向）：

            • 小眼球

            • 浅前房

            • 窄房角

            • 晶体较厚、相对靠前

        4. 4.

          临床表现

          1. 1.

            分期

          2. 2.

            症状：剧烈眼痛、同侧头痛、视朦、虹视、恶心、呕吐

          3. 3.

            体征：

            • 视力下降：甚至仅留手动或光感

            • 高：眼压升高 ＞50mmHg

            • 大：瞳孔散大--纵椭圆形

            • 红：睫状充血或混合充血

            • 肿：角膜水肿--上皮性水肿、雾状

            • 浅：前房变浅和房角闭塞

            • 虹：虹膜节段性萎缩、角膜后色素沉着

            • 斑：晶体改变--青光眼斑

        5. 5.

          诊断

          • 视力急剧下降

          • 眼压突然升高

          • 角膜严重水肿

          • 瞳孔垂直椭圆形扩大

          • 眼局部明显混合充血

          • 伴剧烈眼痛、头痛

        6. 6.

          鉴别诊断


          		 急性闭角型青光眼
          		 急性虹膜睫状体炎
          		 急性结膜炎

          视力 高度减退 不同程度减退 正常
          充血 混合充血 睫状或混合充血 结膜充血

          		 透明

      5. 5.

        慢性（chronic ACG）

        1. 1.

          特点

          • 发病年龄早

          • 眼压升高慢

        2. 2.

          发病因素

          • 解剖因素：前房较浅、房角较狭窄

          • 瞳孔阻滞

          • 非瞳孔阻滞

        3. 3.

          临床表现

          • 视盘在高眼压的持续作用下，渐渐萎缩，形成凹陷，视野随之发生进行性损害

          • 慢性进展过程与开角型青光眼病程相类似，但其视神经损害的发展更快

        4. 4.

          诊断

          • 周边前房浅

          • 房角中度狭窄，有程度不同的虹膜周边前粘连

          • 眼压中等度升高

          • 虽双眼不是同时发病，但“健眼”已有房角狭窄、周边虹膜前粘连

          • 青光眼性视盘凹陷

          • 青光眼视野缺损

    2. 2.

      开角型青光眼（POAG）

      1. 1.

        特点：

        • 两眼中至少一只眼的眼压持续 ≥21mmHg

        • 典型的青光眼性视神经损害和视野缺损

        • 眼压虽然升高但房角始终是开放的（即房水外流受阻于小梁网-Schlemm 管系统）

      2. 2.

        临床表现

        1. 1.

          症状：无任何自觉症状

        2. 2.

          眼压：24 小时波动较大

        3. 3.

          眼前段：多无明显异常

        4. 4.

          眼底

          • 垂直径 C/D 值(杯盘比，即视杯直径与视盘直径比值) 增大,或形成切迹

          • 双眼 C/D 差值 >0.2

          • 盘周浅表出血

        5. 5.

          视功能：视野缺损--诊断和病情评估指标

      3. 3.

        诊断：两项为阳性，房角检查属开角即可诊断

        • 眼压升高（≥21mmHg）

        • 视盘损害

        • 视野缺损

    3. 3.

      治疗

      1. 1.

        目的：保存视功能

      2. 2.

        方法：

        • 降低眼压：迅速控制眼压，减少组织损害（急诊处理）➡️ 择期手术防止再发

        • 视神经保护性治疗

      3. 3.

        常用降压药

        • 药物降眼压途径：① 增加房水流出（最符合生理）;② 抑制房水生成;③ 减少眼内容积

        • 拟副交感神经药(缩瞳剂）：毛果云香碱--缩小瞳孔、开放房角

        • β-肾上腺能受体阻滞剂：噻吗洛尔、贝他洛尔--抑制房水生成，不影响瞳孔大小

        • 肾上腺能受体激动剂：α2 受体激动剂--溴莫尼定--减少房水生成，促进房水流出

        • 前列腺素衍生物：拉坦前列素--只促进房水排出

        • 碳酸酐酶抑制剂：减少房水生成

        • 高渗剂：甘露醇--迅速提高血浆渗透压，使眼组织，特别是玻璃体中的水分进入血液

      4. 4.

        PACG 的治疗

        1. 1.

          缩小瞳孔：1% 毛果芸香碱眼液

        2. 2.

          联合用药：全身应用高渗剂、碳酸酐酶抑制剂，局部滴用 β-受体阻滞剂以迅速降低眼压

        3. 3.

          辅助治疗

        4. 4.

          手术治疗

      5. 5.

        POAG 的治疗

        1. 1.

          药物治疗

          • 前列腺素衍生物

          • β-受体阻滞剂或肾上腺能受体激动剂联合缩瞳剂

        2. 2.

          激光治疗：选择性激光小梁成型术（SLT）

        3. 3.

          滤过性手术：小梁切除术

  3. 3.

    继发性青光眼

    1. 1.

      继发性青光眼：由于某些眼病或全身疾病，干扰或破坏了正常的房水循环， 使房水流出通路受阻而引起眼压增高的一组青光眼

    2. 2.

      青光眼睫状体炎综合征(glaucomatocyclitic syndrom)--青睫综合征（PSS）：最初是 Posner 和 Schlossmann 描述的，因而本病又称 Posner-Schlossman 综合征。是一种反复发作的单眼青光眼合并睫状体炎，好发于中年男性，其特点为单侧，反复发作，视力轻度减退，眼压中等升高，房角开放，少量灰白色 KP。

    3. 3.

      糖皮质激素性青光眼：指长期局部或全身使用糖皮质激素（GC）引起的继发性开角型青光眼，多数停药后可恢复，临床表现与 POAG 相似。

    4. 4.

      眼外伤所致的继发性青光眼：红细胞堆积在小梁网上,或同时伴有血凝块阻滞瞳孔，以及小梁网损伤后炎性水肿,使房水排出受阻所致。

    5. 5.

      虹膜角膜内皮综合征（ICE)

      • 中青年女性

      • 单眼发病

      • 进行性虹膜萎缩、虹膜痣综合征、Chandle 综合征

  4. 4.

    先天性或发育性青光眼

    1. 1.

      婴幼儿型青光眼：畏光、流泪和眼睑痉挛三大特征

    2. 2.

      青少年性青光眼：临床表现同 POAG
