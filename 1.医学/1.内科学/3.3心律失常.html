3.3心律失常

  1. 1.

    概述

    1. 1.

      心脏电生理

      • 心脏射血顺序：左心房 ➡️ 左心室

      • 电传导系统

        • 窦房结（60～100 次/分）

        • 房室结（40～60 次/分）

        • 左右束支

        • 浦肯野纤维（30～40 次/分）

      • 窦房结的作用：自古华山一条路

        • 延缓：保证心房先收缩，心室后收缩

        • 滤过：保护心室（正常时心室与心房频率相同，心房过快时，使心室慢）

      • 心脏传导系统接受迷走与交感神经的双重调节

      notability2021年11月4日.note

    2. 2.

      分类

      1. 1.

        快慢：急性心律失常、慢性心律失常

      2. 2.

        部位

        • 室性心律失常

        • 室上性心律失常：心房、交界区

      3. 3.

        机制

        1. 1.

          冲动形成异常：

          • 窦性心律失常：① 窦性心动过速；② 窦性心动过缓；③ 窦性心律不齐；④ 窦性停搏

          • 异位心律：

            • 被动：逸搏及逸搏心律

            • 主动：① 期前收缩；② 阵发性心动过速；③ 心房扑动、心房颤动；④ 心室扑动、心室颤动

        2. 2.

          冲动传导异常：

          • 干扰及干扰性房室分离

          • 心脏传导阻滞：① 窦房阻滞；② 房内阻滞；③ 房室阻滞(一、二、三度)；④ 室内阻滞

          • 折返性心律

          • 房室间传导途径异常

    3. 3.

      治疗

      1. 1.

        原则：无症状就观察，有症状先用药，一旦休克就电打

      2. 2.

        药物治疗

        药物类别 药物 特点
        IA（快钠通道阻滞剂） 奎尼丁
        IB 苯妥英钠、利多卡因 室速常用
        IC 普罗帕酮
        II（β-肾上腺素受体阻滞剂） 美托洛尔、普奈洛尔 ➖
        III（钾通道阻滞剂） 胺碘酮 合并基础疾病时常用--副作用少
        IV（慢钙通道阻滞剂） 维拉帕米、地尔硫䓬 ➖

        		阿托品、异丙肾上腺素 窦性心动过缓、房室传导阻滞 ➕
        强心苷类 洋地黄 合并心衰时使用（强心作用、阻滞窦房结）
      3. 3.

        介入治疗

        • 心脏起搏治疗：窦性停搏

        • 植入型心律转复除颤器（ICD）：室速

  2. 2.

    窦性心律失常

    1. 1.

      		 诊断 治疗
      窦性心动过速 心率 >100 次/分
      <3 个大格      		 β 受体阻滞剂（美托洛尔）--交感神经
      钙离子通道阻滞剂（维拉帕米、地尔硫䓬）--窦房结
      美拉地
      窦性心动过缓 心律 <60 次/分
      >5 个小格      		 阿托品--（瞳孔扩大为你，口干舌燥为你、心跳加快为你）
      异丙肾上腺素--加快交感神经、房室结、束支传导
      介入手术：放置起搏器
      快脱肾
    2. 2.

      窦性停搏

      • 在较正常 PP 间期显著长的间期内无 P 波发生，或 P 波与 QRS 波均不出现

      • 心率持久减慢 <50 次/分，心律不齐

      • 治疗：放置起搏器

    3. 3.

      窦房传导阻滞（SAB）

      • 二度 I 型：PP 间期进行性缩短，直至出现一次长 PP 间期，该长 PP 间期等于基本 PP 间期的两倍

    4. 4.

      病态窦房结综合征（SSS）

      1. 1.

        SSS：简称病窦综合征，是由窦房结病变导致功能减退，产生多种心律失常的综合表现

      2. 2.

        病因

        • 纤维化与脂肪浸润、硬化与退行性变 、淀粉样变性

        • 甲减

        • 感染

        • 周围神经、心肌病变、窦房结动脉供血减少

      3. 3.

        临床表现

        • 心动过缓：心、脑等供血不足

        • 心动过速：心悸、心绞痛

      4. 4.

        心电图表现

        • 非药物引起的持续而显著的窦性心动过缓（<50 次/分）

        • 窦性停搏与窦房阻滞

        • 窦房阻滞与房室阻滞并存

        • 心动过缓-心动过速综合征（慢快综合征）：指心动过缓与房性快速型心律失常(心房扑动、心房颤动 或房性心动过速)交替发作

      5. 5.

        治疗：无症状 ➡️ 观察，有症状 ➡️ 植入起搏器

  3. 3.

    房性心律失常

    1. 1.

      房性期前收缩（房早）

      1. 1.

        房早：指起源于窦房结以外心房的任何部位的心房激动

      2. 2.

        临床表现

        • 偶发

        • 心悸（一些病人有胸闷、乏力症状，自觉有停跳感）

      3. 3.

        心电图表现

        • P 波提前发生，与窦性 P 波形态不同

        • PR 间期 >120 毫秒

        • QRS 波群呈室上性

        • 多为不完全代偿间歇

      4. 4.

        治疗：一般无需治疗

    2. 2.

      房性心动过速（房速）

      1. 1.

        房速：连续 3 个或 3 个以上的房早，起源于心房且无需房室结参与维持的心动过速

      2. 2.

        临床表现： 当房室传导比例发生变动时，听诊心律不恒定，第一心音强度变化

      3. 3.

        心电图特征

        1. 1.

          局灶性

          • 心房率通常为 150～200 次/分

          • P 波形态与窦性 P 波不同

          • 当房率加快时可出现二度 I 型或 II 型房室阻滞

          • P 波之间的等电线仍存在（与心房扑动鉴别）

          • 剌激迷走神经不能终止心动过速

          • 发作开始时心率逐渐加速

        2. 2.

          多源性

          • 心房率 100～130 次/分

          • 通常有 3 种或以上形态各异的 P 波，PR 间期各不相同

          • 大多数 P 波能下传心室，心室率不规则（因 P 波过早发生而受阻）

      4. 4.

        治疗

        • 病因与诱因治疗

        • 控制心室率

        • 转复窦性心律

    3. 3.

      心房扑动（房扑）

      1. 1.

        房扑：介于房速和心房颤动之间的快速型心律失常

      2. 2.

        病因：多见于器质性心脏病（如风湿性心脏病、冠心病）

      3. 3.

        临床表现

        • 伴有极快的心室率，可诱发心绞痛与充血性心力衰竭

        • 可产生心房血栓，进而引起体循环栓塞

        • 体格检查可见快速的颈静脉扑动

      4. 4.

        心电图特征

        • 窦性 P 波消失，代之以振幅、间距相同的有规律的锯齿状扑动波，称为 F 波， 扑动波之间的等电线消失 ，频率常为 250～350 次/分

        • 心室率不规则，房扑波多以 2: 1 及 4: 1 交替下传

        • QRS 波形态正常

      5. 5.

        治疗

        • 药物治疗：β 受体阻滞剂、钙通道阻滞剂

        • 非药物治疗：直流电复律是终止房扑最有效的方法，导管消融可根治房扑

        • 抗凝治疗

    4. 4.

      心房颤动（房颤）

      1. 1.

        房颤（AF）：指规则有序的心房电活动丧失，代之以快速无序的颤动波，是严重的心房电活动紊乱

      2. 2.

        心室律(率)紊乱、心功能受损和心房附壁血栓形成是主要病理生理特点

      3. 3.

        病因

        • 心内因素：风湿性心脏病二尖瓣狭窄

        • 心外因素：甲亢

      4. 4.

        分类（口诀：一周年）

      5. 5.

        临床表现

        • 房颤并发血栓栓塞的危险性甚大（以脑栓塞危害最大）--（解题：哪里不舒服，哪里有血栓）

        • 体征（口诀：不不短）

          • 第一心音强度变化不定

          • 心律极不规则

          • 脉搏短绌

      6. 6.

        心电图特征

        • P 波消失，代之以小而不规则的基线波动，形态与振幅均变化不定，称为 f 波； 频率为 350～600 次/分

        • 心室率极不规则，100～160 次/分

        • QRS 波正常

      7. 7.

        治疗

        1. 1.

          原则：一怕“快”，二怕“栓”

        2. 2.

          抗凝治疗：华法林

          • 房颤持续 >24h 应在复律前接受华法林有效抗凝治疗 3 周，待成功复律后继续治疗 4 周（朝 3 暮 4），否则附壁血栓脱落会产生严重后果

        3. 3.

          转复并维持窦性心律

          • 药物复率：胺碘酮--万能药（合并严重基础疾病时使用）

          • 电复率

          • 导管消融治疗

        4. 4.

          控制心室率：β 受体阻滞剂、钙通道阻滞剂

  4. 4.

    房室交界区性心律失常

    1. 1.

      交界性期前收缩

      • 提前发生的 QRS 波群与逆行 P 波

      • 逆行 P 波可位于 QRS 波群之前（PR 问期 <0. 12 秒）、之中或之后（RP 间期 <0. 20 秒）

    2. 2.

      阵发性室上性心动过速

      1. 1.

        多数患者无器质性心脏病，共同发病机制为折返机制

      2. 2.

        房室结折返性心动过速（AVNRT）

        1. 1.

          临床表现：发作突然起始与终止，持续时间长短不一（突发突止）

        2. 2.

          心电图特征：

          • 心率 150～250 次/分，节律规则

          • QRS 波形态与时限均正常

          • P 波为逆行性（II、III、aVF 导联倒置），常埋藏于 QRS 中

        3. 3.

          治疗

          • 刺激迷走神经手法：颈动脉窦按摩、Valsava 动作

          • 药物治疗：首选腺苷，切忌应用阻断房室结的药物（洋地黄、 β 受体阻滞剂、异搏定等）

          • 病情严重、药物无效时--直流电复律

          • 导管消融技术--最好方法，可治愈

      3. 3.

        房室折返性内心动过速（AVRT）与预激综合征

        1. 1.

          预激综合征（preexcitation syndrome）：是指心房部分激动由正常房室传导系统以外的先天性附加通道(旁道)下传，使心室某一部分心肌预先激动(预激)，导致以异常心电生理和(或)伴发多种快速型心律失常为特征的一种综合征

        2. 2.

          由 Kent 束引起的心室预激并伴有快速型心律失常者称为典型预激综合征，又称为 WPW 综合征

        3. 3.

          心电图表现

          • QRS 波形态多样

        4. 4.

          治疗同上

  5. 5.

    室性心律失常

    1. 1.

      室性期前收缩（室早）

      1. 1.

        心电图表现

        • 提前发生的 QRS 波群，时限常超过 0.12 秒、宽大畸形

        • ST 段与 T 波的方向与 QRS 主波方向相反

        • 室性期前收缩与其前面的窦性搏动之间期(称为配对间期)恒定

      2. 2.

        治疗

        1. 1.

          无明显症状或症状轻微者，不必药物治疗

        2. 2.

          需要紧急处理的室性期前收缩：

          • 频发室性期前收缩（每分钟超过 5 次）

          • 多源室性期前收缩

          • 成对或连续出现的室性期前收缩

          • 室性期前收缩落在前一个心搏的 T 波上（R on T）

        3. 3.

          急性心肌缺血或梗死合并室性期前收缩病人，首选再灌注治疗

    2. 2.

      室性心动过速（室速）

      1. 1.

        连续 3 个或 3 个以上的室早，属于恶性心律失常

      2. 2.

        病因：室速常发生于各种器质性心脏病病人

      3. 3.

        临床表现：非持续性室速常无症状；持续性室速常伴有明显血流动力学障碍与心肌缺血

      4. 4.

        心电图表现

        • 3 个或以上的室性期前收缩连续出现

        • 心室率常为 100～250 次/分

        • 心房独立活动与 QRS 波无固定关系，形成室房分离

        • 可见心室夺获（窦房结偶夺回控制权）与室性融合波

      5. 5.

        治疗：

        • 利多卡因、β-受体阻滞剂或胺碘酮静脉推注

        • 休克、低血压等直流电复律

        • 植入心律转复除颤器 (ICD)

      6. 6.

        尖端扭转型室速（TDP）治疗首先给予静脉注射镁盐

    3. 3.

      心室扑动与心室颤动（室扑、室颤）

      1. 1.

        为致死性心律失常，常见于缺血性心脏病

      2. 2.

        心电图特征

        • 心室扑动呈正弦图形，波幅大而规则， QRS 波呈单形性，频率 150 ~ 300 次/分

        • 心室颤动的波形、振幅与频率均极不规则，无法辨认 QRS 波群、ST 段与 T 波，持续时间较短，如不及时抢救，一般心电活动在数分钟内迅速消失

        • 急性心肌梗死由于舒张早期的室性期前收缩落在 T 波上触发室速 (R-on-T)，后演变为室颤

      3. 3.

        临床表现：突然意识丧失，抽搐，呼吸停止甚至死亡，无血压、脉搏，无心音

      4. 4.

        治疗：非同步电除颤：① 单向波：360J；② 双向波：150～200J

  6. 6.

    心脏传导阻滞

    1. 1.

      房室阻滞

      1. 1.

        心电图特征（离婚三部曲）

        1. 1.

          一度房室阻滞：PR 间期超过 0.20 秒

        2. 2.

          二度房室阻滞

          • I 型（文氏阻滞）：PR 间期逐渐延长，直到 P 波下传受阻，脱漏 1 个 QRS 波群

          • II 型（莫氏阻滞）：PR 间期恒定，部分 P 波后无 QRS 波群

        3. 3.

          三度房室阻滞：P 波与 QRS 波群各自成节律、互不相关（听诊大炮音）

      2. 2.

        治疗

        • 阿托品、异丙肾上腺素

        • 对于症状明显、心室率缓慢者，应及早给予临时性或永久性心脏起搏治疗

    2. 2.

      室内阻滞

      1. 1.

        临床表现

        • 右束支阻滞较为常见，正常人亦可发生

        • 左束支阻滞多见于器质性心脏病患者

      2. 2.

        心电图特征

        1. 1.

          共同特征

          • QRS≥0.12 秒

          • ST-T 改变

        2. 2.

          右束支阻滞（RBBB）

          • V1 呈 rsR'

          • V5、V6 导联呈 qRS 或 RS，S 波宽阔

        3. 3.

          左束支阻滞（LBBB）

          • V1、V2 导联呈宽阔的 QS 波或 rS 波形，S 波宽大

          • V5、V6 导联 R 波宽大，顶部有切迹或粗钝

      3. 3.

        治疗：慢性单侧束支阻滞的病人如无症状，无需接受治疗
