2.10.肺血栓栓塞症

  • 肺栓塞( pulmonary embolism)：是以各种栓子阻塞肺动脉或其分支为其发病原因的一组疾病或临床综合征的总称

  • 引起PTE(肺血栓栓塞症)的血栓主要来源于DVT(深静脉血栓形成)，两者合称为VTE(静脉血栓栓塞症)

流行病学：检出率低，漏诊和误诊严重

危险因素：

  • Virchow三要素：任何可以导致①静脉血液淤滞、②静脉系统内皮损伤和③血液高凝状态的因素

  • 年龄是独立的危险因素

一、病理生理

  • 引起PTE的血栓多来源于下肢深静脉

  • 多部位或双侧性的血栓栓塞更为常见

  • 影像学发现栓塞更易发生于右侧和下肺叶

  1. 1.

    血流动力学改变

  2. 2.

    气体交换障碍：呼吸功能不全，出现低氧血症和代偿性过度通气 (低碳酸血症)或相对性肺泡低通气

  3. 3.

    肺梗死：肺组织同时接受肺动脉、支气管动脉和肺泡内气体三重氧供，故梗死概率低

  4. 4.

    慢性血栓栓塞性肺动脉高压(CTEPH)：指急性PTE后肺动脉内血栓未完全溶解，或PTE反复发生出现的症状

二、临床表现

  1. 1.

    症状

    1. 1.

      不明原因的呼吸困难及气促，尤以活动后明显(最多见)

    2. 2.

      胸痛

    3. 3.

      晕厥（可为PTE的唯一或首发症状）

    4. 4.

      烦躁不安、惊恐甚至濒死感

    5. 5.

      咯血（肺缺血性坏死）

    6. 6.

      咳嗽、心悸

  2. 2.

    体征

    1. 1.

      呼吸系统：呼吸急促

    2. 2.

      循环系统：①心动过速，血压变化；②肺动脉瓣区第二音亢进( P2>A2)或分裂

  3. 3.

    DVT的症状和体征

    1. 1.

      患肢肿胀、周径增粗，行走后患肢易疲劳或肿胀加重

    2. 2.

      大、小腿周径双侧相差>1cm即有临床意义

三、诊断

  1. 1.

    疑诊

    1. 1.

      血浆D-二聚体：对血栓形成具有很高的敏感性，含量正常，则对 PTE有重要的排除诊断价值

    2. 2.

      动脉血气分析：低氧血症、低碳酸血症

    3. 3.

      心电图：

      • 窦性心动过速

      • 可出现甚或的T波倒置和 ST段异常、征

    4. 4.

      X线胸片:

      1. 1.

        肺动脉阻塞征

      2. 2.

        肺动脉高压征及右心扩大征

      3. 3.

        肺组织继发改变：尖端指向肺门的楔形影

    5. 5.

      超声心动图，满足两项即可诊断右心功能障碍：

      1. 1.

        右心室扩张

      2. 2.

        右心室壁运动幅度减低

      3. 3.

        吸气时下腔静脉不萎陷

      4. 4.

        三尖瓣反流压差 >30mmHg

    6. 6.

      下肢深静脉检查：超声为最简便方法

  2. 2.

    确诊(一项阳性即可诊断)

    1. 1.

      CT肺动脉造影(CTPA)

      1. 1.

        直接征象：肺动脉充盈缺损

      2. 2.

        间接征象：肺野楔形密度增高影

    2. 2.

      放射性核素肺通气/血流灌注 (V/Q) 显像：呈肺段分布的肺血流灌注缺损，并与通气显像不匹配

    3. 3.

      磁共振成像和磁共振肺动脉造影(MRI/MRPA):可以直接显示肺动脉内的栓子及PTE所致的低灌注区

    4. 4.

      肺动脉造影：是PTE诊断的“金标准”

  3. 3.

    求因

    1. 1.

      明确有无DVT：疑诊PTE均应进行下肢深静脉加压超声检查

    2. 2.

      寻找发生DVT和PTE的诱发因素：制动、创伤、肿瘤、长期口服避孕药、易栓倾向

四、临床分型

  1. 1.

    高危PTE：以休克和低血压为主要表现

    • 判断标准：体循环动脉收缩压<90mmHg，或较基础值下降幅度≥40mmHg，持续15分钟以上

  2. 2.

    中危PTE：血流动力学稳定，但存在右心功能不全和(或)心肌损伤

    • 心肌损伤判断：心电图ST段升高或压低，或T波倒置

  3. 3.

    低危PTE：血流动力学稳定 ，无右心功能不全和心肌损伤

五、治疗

  1. 1.

    一般治疗

    • 卧床休息，保持大便通畅，避免用力，以免深静脉血栓脱落

  2. 2.

    抗凝治疗：

    • 临床疑诊，无禁忌证时，应开始抗凝治疗

    • 对于栓子来源不明的首发病例，需至少给予 6 个月的抗凝

    1. 1.

      普通肝素：注意监测血小板，血小板持续降低达50%以上，和(或)出现动、静脉血栓的征象，应停用

    2. 2.

      低分子量肝素

    3. 3.

      磺达肝癸钠：无HIT作用，可用于VTE的初始治疗

    4. 4.

      华法林：

      • 与肝素类药物需至少重叠应用 5 天，当国际标准化比值(INR)达到 2. 5(2. 0 ~ 3. 0)，持续至少 24 小时，方可停用

      • 一般口服华法林的疗程至少为3个月

    5. 5.

      其他抗疑药物：阿加曲班、比伐卢定（主要用于发生 HIT的病人）

  3. 3.

    溶栓治疗：

    • 主要适用于高危PTE

    • 溶栓的时间窗一般定为14 天以内

    • 溶栓应尽可能在 PTE 确诊的前提下慎重进行

    • 绝对禁忌证：活动性内出血和近期自发性颅内出血

    • 常用的潜栓药物：尿激酶(UK)、链激酶(SK)和重组组织型纤溶酶原激活剂(rt-PA)

      • 链激酶具有抗原性，用药前需肌内注射苯海拉明或地塞米松，以防止过敏反应

      • 链激酶 6个月内不宜再次使用

    • 溶栓治疗后，应每 2 ~4 小时测定一次APTT，当其水平降至正常值的 2倍(≤60 秒)时 ，即应启动规范的肝素治疗

  4. 4.

    肺动脉导管碎解和抽吸血栓：

    适用于肺功脉主干或主要分支的高危PTE并且：

    • 溶栓治疗禁忌

    • 经溶栓或积极的内科治疗无效

    • 在溶栓起效前(在数小时内)很可能会发生致死性休克

  5. 5.

    肺动脉血栓摘除术

  6. 6.

    放置腔静脉滤器

  7. 7.

    CTEPH的治疗：长期口服华法林抗凝治疗，维持INR2～3
