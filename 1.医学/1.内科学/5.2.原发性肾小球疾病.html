5.2.原发性肾小球疾病

  1. 1.

    概述

    1. 2.

      肾小球疾病：是一组以血尿、蛋白尿、水肿、高血压、肾功能损害等为主要临床表现，病变通常累及双侧肾小球的常见疾病。

    2. 3.

      发病机制：免疫介导性炎症疾病

      • 免疫反应

        • 体液免疫：循环免疫复合物沉积、原位免疫复合物形成、自身抗体

        • 细胞免疫

      • 炎症反应：呼吸道

      • 非免疫因素：肾小球毛细血管内高压力、蛋白尿、高脂血症

    3. 4.

      临床表现

      1. 1.

        蛋白尿：

        • 尿蛋白：>150mg/d➡️ 蛋白尿；>3.5g/d➡️ 大量蛋白尿

        • 电荷屏障或分子屏障的损伤均可引起蛋白尿，肾小球性蛋白尿常以白蛋白为主

      2. 2.

        血尿：其血尿常为无痛性、全程性血尿


        		 肾小球源性
        		 非肾小球源性
        新鲜尿沉渣相差显微镜检查 变形红细胞 均一形态正常红细胞
        尿红细胞容积分布曲线 非对称曲线
        峰值红细胞容积 < 静脉峰值红细胞容积
        		 对称曲线
        峰值红细胞容积 > 静脉峰值红细胞容积
      3. 3.

        水肿


        		 肾病性水肿
        		 肾炎性水肿
        机制 蛋白尿--> 血浆胶体渗透压下降--> 液体渗入组织间隙 肾小球滤过率下降--> 球管失衡--> 水钠潴留
        表现 水肿多从下肢部位开始 水肿多从眼险、颜面部开始
      4. 4.

        高血压

        • 水钠潴留

        • 肾素分泌增多

        • 肾内降压物质分泌减少

      5. 5.

        肾功能损害

    4. 5.

      单纯型肾病（simple type NS）和肾炎型肾病（nephritic type NS）的鉴别，凡具有以下 4 项之一或多项者属于肾炎型肾病：

      • 2 周内分别 3 次以上离心尿检查红细胞 ≥10 个/HP,并证实为肾小球源性血尿者

      • 反复或持续高血压，并除外糖皮质激素等原因所致

      • 肾功能不全，并排除由于血容量不足等所致

      • 持续低补体血症

    5. 6.

      治疗原则：

      • 去除病因

      • 抑制免疫反应及炎症反应

      • 对症治疗，防治并发症

      • 延缓肾脏疾病进展和肾脏替代治疗

  2. 2.

    肾病综合征

    1. 1.

      肾病综合征（NS）：是一组由多种原因引起的肾小球基底膜通透性增加，导致血浆内大量蛋白质从尿中丢失的临床综合征。临床有四大特点

    2. 2.

      病因


      		 儿童 青少年 中老年
      原发性 微小病变型肾病 系膜增生性肾小球肾炎 膜性肾病
      继发性 过敏性紫癜肾炎 狼疮肾炎 糖尿病肾病
    3. 3.

      病理生理

      1. 1.

        大量蛋白尿：分子屏障及电荷屏障受损

      2. 2.

        低白蛋白血症：肝脏白蛋白合成增加不足以克服丢失和分解 ➡️ 长期导致营养不良、生长发育迟缓

      3. 3.

        水肿：血浆胶体渗透压下降，使水分从血管腔内进入组织间隙

        • 血浆白蛋白 <25g/L➡️ 液体在间质区滞留（水肿）

        • 血浆白蛋白 <15g/L➡️ 腹腔、胸腔积液形成

      4. 4.

        高脂血症：肝脏脂蛋白合成的增加和外周组织利用及分解减少

      5. 5.

        其他

        • 体液免疫功能降低：血清 IgG 和补体系统 B、D 因子从尿中大量丢失

        • 高凝状态：抗凝血酶 III 丢失，而 IV、V、VII 因子和纤维蛋白原增多

        • 结合钙、游离钙降低：钙结合蛋白丢失、25-(OH)D3 结合蛋白丢失

        • 小细胞低色素贫血：转铁蛋白减少

    4. 4.

      病理类型及临床特征


      		 特征 治疗
      微小病变型肾病（MCNS） 电镜下广泛肾小球脏层上皮细胞足突融合 激素敏感
      系膜增生性肾小球肾炎（MSPGN）
      		 光镜下肾小球系膜细胞和系膜基质弥漫增生 激素、细胞毒敏感
      局灶节段性肾小球硬化（FSGS） 受累节段硬化，相应的肾小管萎缩、肾间质纤维化 激素不敏感，平均缓解期 4 个月
      膜性肾病（MN） 光镜下钉突形成(嗜银染色) 易发血栓栓塞并发症
      系膜毛细血管性肾小球肾炎 毛细血管袢呈“双轨征”
      其他男多于女，该病男女比例相当      		 常在上呼吸道感染后出现急性肾炎综合征
    5. 5.

      诊断

      1. 1.

        明确是否为肾病综合征（三高一低，前两项为必备条件）

        • 大量蛋白尿（>3.5g/d）

        • 低白蛋白血症（血清白蛋白 <30g/L）

        • 水肿

        • 高脂血症

      2. 2.

        确认病因

      3. 3.

        判断有无并发症

    6. 6.

      治疗

      1. 1.

        一般治疗

        1. 1.

          休息：有严重水肿、低白蛋白血症者需卧床休息，水肿消失、一般情况好转后，可起床活动

        2. 2.

          水肿时应低盐（＜3g/d）饮食，不可长期忌盐

        3. 3.

          正常量 0.8～1.0g/（kg·d）的优质蛋白（富含必需氨基酸的动物蛋白）饮食

        4. 4.

          保证足够的热量（30～35Kcal/Kg.d)

        5. 5.

          少进富含饱和脂肪酸（动物油脂）的饮食，而多吃富含多聚不饱和脂肪酸（如植物油、鱼油），进食富含可溶性纤维（如燕麦、米糠及豆类）的饮食

      2. 2.

        对症治疗：不宜过快、过猛，以免造成有效血容量不足、加重血液高黏倾向，诱发血栓、栓塞并发症

        • 利尿消肿：

        • 减少尿蛋白：血管紧张素转换酶抑制剂（ACEI）或血管紧张素 II 受体阻滞剂（ARB）

      3. 3.

        免疫抑制治疗：

        1. 1.

          糖皮质激素：

          • 原则：① 起始足量(1mg/kg·d，8～12w) ② 缓慢减药 ③ 长期维持

          • 疗效评价：

            • 激素敏感型(用药 8～12 周内缓解)

            • 激素依赖型(激素减到一定程度即复发)

            • 激素抵抗型(激素治疗无效)

        2. 2.

          细胞毒药物：用于激素依赖或激素无效型病人，协同激素治疗

          • 环磷酰胺：目前最常用，有较强的免疫抑制作用

            • 主要不良反应：骨髓抑制、肝损害、性腺抑制（男性）

          • 苯丁酸氮介

        3. 3.

          钙调神经蛋白抑制剂：作为二线药物用于治疗激素及细胞毒药物无效的难治性肾病综合征

          • 环孢素（CsA）：

          • 他克莫司（FK506）：但肾毒性副作用小于 CsA

        4. 4.

          吗替麦考酚酯（MMF）：已广泛用于肾移植后排斥反应，不良反应少，但费用昂贵

      4. 4.

        预防并发症

        • 感染：上呼吸道感染最常见（病毒感染常见）

        • 血栓和栓塞：血浆白蛋白低于 20g／L 即应开始抗凝治疗；发生血栓、栓塞者尽早给予溶栓治疗，治疗时避免药物过量导致出血

        • 急性肾损伤：① 袢利尿剂；② 血液透析；③ 原发病治疗；④ 碱化尿液

        • 肾小管功能损害

        • 蛋白质及脂肪代谢紊乱：① 调整饮食中蛋白和脂肪的量与结构；② 药物

        • 电解质紊乱和低血容量：低钠、低钾及低钙血症

  3. 3.

    急性肾小球肾炎（AGN）

    1. 1.

      AGN：是指一组病因不一，临床表现为急性起病，多有前驱感染，以血尿为主，伴不同程度蛋白尿，可有水肿、高血压，或肾功能不全等特点的肾小球疾病

    2. 2.

      病因和发病机制：β-溶血性链球菌“致肾炎菌株”感染诱发免疫反应所致

    3. 3.

      病理表现

      • 肾脏体积增大

      • 光镜下见弥漫性肾小球毛细血管内皮细胞及系膜细胞增生（毛细血管内增生性肾小球肾炎）

      • 肾间质水肿及灶状炎症细胞浸润

      • 免疫荧光 IgG 和 C3 呈粗颗粒状沉积

      • 电镜见肾小球上皮细胞下有驼峰状电子致密物沉积

    4. 4.

      临床表现：

      1. 1.

        表现

      2. 2.

        特点

        • 儿童多见，男 > 女

        • 有前驱感染史，呼吸道及皮肤感染为主（发病前 1～3W）

        • 急性起病，病情轻重不一

        • 预后好，有自限性

      3. 3.

        严重表现

        • 严重循环充血

        • 高血压脑病

        • 急性肾功能不全

      4. 4.

        非典型表现

        • 无症状性急性肾炎

        • 肾外症状性急性肾炎

        • 以肾病综合征为表现的急性肾炎

    5. 5.

      诊断：

      • 链球菌感染后 1～3W，出现血尿、蛋白尿、水肿、高血压、一过性肾功能损害

      • 起病初期血清 C3 及总补体下降，8 周内逐渐恢复正常

      • 病理类型：毛细血管内增生性肾小球肾炎

      • 血清抗链球菌溶血素“O”（ASO）滴度升高，提示链球菌感染

      • 肾活检指征：

        • 少尿一周以上或进行性尿量减少伴肾功能恶化

        • 病程超过 8 周而无好转趋势者

        • 急性肾炎综合征伴肾病综合征者

    6. 6.

      治疗

      • 休息：急性期需卧床 2~3 周，直到肉眼血尿消失，水肿减退，血压正常，即可下床进行轻微活动

      • 饮食：低盐、优质低蛋白饮食

      • 抗感染：有感染灶时用青霉素 10〜14 天

      • 对症治疗：

        • 利尿：氢氯噻嗪

        • 降血压：硝苯地平、卡托普利

      • 一般不主张用激素和细胞毒药物（自限性）

      • 并发症治疗

        • 严重循环充血：纠正水钠潴留；肺水肿 ➡️ 硝普钠

        • 高血压脑病的治疗：硝普钠（降血压效力强而迅速）

  4. 4.

    急进性肾小球肾炎（RPGN）

    1. 1.

      RPGN：在急性肾炎综合征基础上，肾功能快速进展，病理类型为新月体肾炎的一组疾病，以急性肾炎综合征、肾功能急剧恶化、早期出现少尿性急性肾功能衰竭为临床特点

    2. 2.

      病理

      • 肾脏体积增大

      • 病变早期为细胞新月体，后期为纤维新月体--无法挽救

      • 分型：


        		 特点
        		 诊断
        I 型 抗肾小球基底膜(GBM)型 抗 GBM 抗体 ➕
        II 型 免疫复合物型 血液循环免疫复合物（CIC）➕
        冷球蛋白 ➕
        III 型 少免疫沉积型（与 ANCA相关小血管炎相关） 抗中性粒细胞胞浆抗体（ANCA）➕
    3. 3.

      临床表现

      • 我国以 II 型略为多见

      • 起病急，病情急骤进展。在急性肾炎综合征基础上，早期出现少尿或无尿，进行性肾功能恶化并发展成尿毒症

      • 常伴有中度贫血

      • II 型伴肾病综合征，III 型有系统性血管炎表现

      • 病人多有前驱上呼吸道感染病史；诱发因素：接触有机化学溶剂、碳氢化合物如汽油

    4. 4.

      诊断：

      • 急性肾炎综合征

      • 进行性肾功能下降

      • 病理（新月体肾炎）

      • 实验室检查

    5. 5.

      治疗

      1. 1.

        强化治疗：

        • 强化血浆置换疗法：适用于 Ⅰ、III 型，应早期应用

        • 甲泼尼龙冲击（静脉滴注） + 环磷酰胺：主要适用于 II、III 型

      2. 2.

        支持对症治疗

        • 及时透析

        • 肾移植（病情静止半年后）

  5. 5.

    慢性肾小球肾炎（CGN）

    1. 1.

      CGN：以蛋白尿、血尿、高血压和水肿为基本临床表现，起病方式各有不同，病情迁延并呈缓慢进展，可有不同程度的肾功能损害，部分病人最终将发展至终末期肾衰竭。

    2. 2.

      病因和发病机制：绝大多数由不同病因的原发性肾小球疾病发展而来，仅少数由急性肾炎发展所致

    3. 3.

      病理：多种病理类型，最终转化成硬化性肾小球肾炎（肾小球硬化、肾小管萎缩、肾间质纤维化）

    4. 4.

      临床表现：

      • 中青年为主

      • 起病缓慢、隐袭，有急性发作倾向

      • 最终发展为慢性肾功能衰竭

      • 部分病人血压持续性中等以上程度升高

      • 可因感染、劳累、血压增高或用肾毒性药物而急剧恶化

    5. 5.

      实验室检查

      • 尿异常：蛋白尿（1-3g/d）、红细胞 ⬆️、可见管型

      • 肾功能异常：BUN，Scr 增高

    6. 6.

      诊断：尿化验异常，伴或不伴水肿及高血压病史达 3 个月以上

    7. 7.

      治疗：

      • 治疗目的：

        • 防止或延缓肾功能进行性恶化

        • 改善或缓解临床症状

        • 防治心脑血管并发症

      • 限制食物中蛋白及磷摄入量：优质低蛋白饮食

      • 积极控制高血压和减少尿蛋白

        • 目标：BP<130/80mmHg，尿蛋白 <1g/d

        • 药物：ACEI、ARB、CCB（具有肾脏保护作用）

      • 糖皮质激素和细胞毒药物：根据病理类型、尿蛋白、肾功能等决定

      • 避免加重肾损害的因素：感染、劳累、妊娠、肾毒性药物

  6. 6.

    无症状性血尿和(或)蛋白尿

    1. 1.

      无症状性血尿和(或)蛋白尿：仅表现为肾小球源性血尿和（或）轻至中度蛋白尿,不伴水

      肿、高血压及肾功能损害的一组肾小球疾病，通常通过实验室检查发现并诊断

    2. 2.

      病理：可由多种病理类型的原发性肾小球疾病所致，但病理改变多较轻

    3. 3.

      临床表现：

      • 临床多无症状

      • 常因发作性肉眼血尿或体检提示镜下血尿或蛋白尿而发现

      • 无水肿、高血压和肾功能损害

      • 部分病人可于高热或剧烈运动后出现一过性血尿，短时间内消失

    4. 4.

      实验室检查：

      • 镜下血尿、蛋白尿(0.5g/24h< 尿蛋白 <2.0g/24h ,以白蛋白为主）

      • 免疫学检查正常

      • 肾功能及影像学检查正常

      • 随访中出现血尿、蛋白尿加重和（或）肾功能恶化，应尽快做肾活检明确诊断

    5. 5.

      诊断

      • 肾小球源性血尿、蛋白尿

      • 无水肿、高血压及肾功能减退

      • 除外其他肾小球疾病，必要时需依赖肾活检方能确诊

    6. 6.

      治疗原则

      • 定期检查和追踪患者（每 3～6 个月 1 次）

      • 保护肾功能、避免肾损伤的因素

      • 对伴血尿的蛋白尿病人，考虑使用 ACEI/ARB 类药物治疗

      • 合并慢性扁桃体炎反复发作，可待急性期过后行扁桃体切除术

      • 随访中如出现高血压或肾功能损害，按慢性肾小球肾炎治疗

      • 可适当用中医药辨证施治，但需避免肾毒性中药

  7. 7.

    IgA 肾病

    1. 1.

      IgA 肾病：是肾小球系膜区以 IgA 或 IgA 沉积为主的原发性肾小球，是我国最常见的肾小球疾病类型，也是终末期肾病的重要病因。

    2. 2.

      发病机制：感染等二次“打击”刺激自身抗体的产生，免疫复合物形成并沉积于肾小球产生炎症反应

    3. 3.

      病理：

      • 弥漫性肾小球系膜细胞和基质增生

      • 病理变化多种多样，程度轻重不一，可涉及几乎所有类型

      • 免疫病理以 IgA 在系膜区沉积为主，电镜见系膜区电子致密物沉积

      • 牛津分型：

        • 系膜细胞增生（M0/1 )

        • 内皮细胞增生（E0/1)

        • 节段性硬化或粘连（S0/1)

        • 肾小管萎缩或肾间质纤维化（T0/1/2)

        • 细胞或细胞纤维性新月体(C0/1/2)

    4. 4.

      临床表现

      • 起病隐匿，常表现为无症状性血尿（以肾小球源血尿为主的混合性血尿）

      • 上呼吸道或消化道感染等前驱症状，与肉眼血尿发作间期短（数小时，3 天内）

      • 血清 IgA 增高，补体水平正常

    5. 5.

      诊断标准：

      • 与上呼吸道感染有关的血尿和（或）蛋白尿

      • 肾活检免疫病理：肾小球系膜区或伴有毛细血管壁 IgA 为主的免疫球蛋白呈颗粒样或团块状沉积

    6. 6.

      治疗：

      • 单纯镜下血尿：一般无需特殊治疗

      • 反复发作性肉眼血尿：控制感染，去除病灶（无肾毒性抗生素--青霉素）

      • 伴高血压、蛋白尿：ACEI、ARB

      • 肾病综合征：病理改变轻者可试用激素及环磷酰胺

      • 急性肾衰竭：激素及环磷酰胺冲击治疗、血液透析治疗

  8. 8.

    总结

    1. 1.

      肾脏疾病，抗感染使用青霉素--无肾毒性；治疗蛋白尿、高血压使用 ACEI、ARB；激素和细胞毒素的使用需谨慎
