2.7.肺结核

  1. 1.

    流行病学：高感染率，高患病率，高耐药率

  2. 2.

    结核分枝杆菌

    1. 1.

      染色：抗酸染色呈红色，可抵抗盐酸酒精的脱色作用，称抗酸杆菌

    2. 2.

      杀菌：干燥、冷、 酸、碱等抵抗力强；紫外线敏感(太阳光2～7h,紫外线灯30min)

    3. 3.

      成分：

      • 类脂质：组织坏死、干酪液化、空洞、结核变态反应

      • 蛋白质：皮肤变态反应

      • 多糖类：血清反应

  3. 3.

    传播

    1. 1.

      传染源：结核病人（痰直接涂片阳性者）

    2. 2.

      传播途径：飞沫传播

    3. 3.

      易感人群：婴幼儿、老年人、HIV感染者、免疫抑制剂使用者、慢性疾病病人

    4. 4.

      传染性大小：

      • 菌量

      • 空间含结核分枝杆菌微滴密度

      • 通风情况

      • 接触的密切程度和时间长短

      • 个体免疫力状况

  4. 4.

    发生发展

    1. 1.

      原发感染

      1. 1.

        是否感染取决于：结核分枝杆菌的毒力和肺泡内巨噬细胞固有的吞噬杀菌能力

      2. 2.

        结核病主要的免疫保护机制是细胞免疫

      3. 3.

        Koch现象：机体对结核分枝杆菌再感染和初感染所表现出不同反应的现象

      4. 4.

        原发综合征

    2. 2.

      继发性结核

      1. 1.

        临床特点：容易出现空洞和排菌，有传染性（所以是临床防控的重点）

      2. 2.

        分类：

        • 发病慢，临床症状少而轻，预后好

        • 发病快，发现时已出现广泛的病变、空洞和播散，痰涂片检查阳性（免疫力低下者易患）

      3. 3.

        发病方式：

        • 内源性复发（占大部分）

        • 外源性重染

  5. 5.

    病理：基本病理变化：渗出、增生、干酪样坏死（较严重，多发生于菌毒力强、免疫力低下时）

  6. 6.

    临床表现：

    1. 1.

      症状：咳嗽、咳痰两周以上或痰中带血、胸膜性胸痛；倦怠乏力、盗汗、食欲减退和体重减轻

    2. 2.

      体征：

      病变 体征
      渗出性病变范围大或干酪样坏死 肺实变体征
      较大的空洞性病变 支气管呼吸音
      纤维条索形成 气管向患侧移位，患侧胸廓塌陷
      结核性胸膜炎 胸腔积液体征
  7. 7.

    诊断：

    1. 1.

      诊断方法

      1. 1.

        病史和症状体征：①症状体征情况②诊断治疗过程（新发、复发）③肺结核接触史

      2. 2.

        影像学检查：

        • 方法：正、侧位胸片

        • 易发部位：上叶的尖后段、 下叶的背段和后基底段

      3. 3.

        痰结核分枝杆菌检查：确诊主要方法，可疑病人必须查痰

        1. 1.

          标本收集：痰三杯（清晨痰、夜间痰和即时痰）

        2. 2.

          方法：

          • 痰涂片检查：简单、快速、易行、可靠，但欠敏感（只能说明痰液中含抗酸杆菌）

          • 培养法：结核病诊断的“金标准”，但培养费时较长，一般为 2 ~8 周

          • 比例法：为药物敏感性测定的“金标准”

      4. 4.

        纤维支气管镜：可在病灶部位钳取活体组织进行病理学检查和结核分枝杆菌培养

      5. 5.

        结核菌素试验：不能区分是结核分枝杆菌的自然感染还是卡介苗接种的免疫反应

      6. 6.

        γ-干扰素释放试验：IGRAs

    2. 2.

      诊断程序

    3. 3.

      分类

      1. 1.

        原发型肺结核

        • 结核菌素试验多为强阳性，

        • 原发综合征：X 线胸片表现为哑铃型阴影，即原发病灶、引流淋巴管炎和肿大的肺门淋巴结

      2. 2.

        血行播散型肺结核

        • 急性粟粒型肺结核：由肺尖至肺底呈大小、密度和分布三均匀的粟粒状结节阴影

        • 亚急性、慢性血行播散型肺结核：双上、中肺野为主的大小、密度和分布三不均匀的粟粒状或结节状阴影

      3. 3.

        继发性肺结核

        • 浸润性肺结核：多发生在肺尖和锁骨下，影像学表现为小片状或斑点状阴影

        • 空洞性肺结核：痰中经常排菌

        • 干酪性肺炎

        • 纤维空洞性肺结核：病程长反复进展恶化；肺门抬高和肺纹理呈垂柳样；纵隔向患侧移位

        • 结核球：周边多有卫星灶

      4. 4.

        结核性胸膜炎：turn to chapter12

      5. 5.

        其他肺外结核：骨关节结核、肾结核、肠结核等

      6. 6.

        菌阴肺结核：三次痰涂片及一次培养均阴性的肺结核

        判断标准：

        • 满足3项

          • 典型肺结核临床症状和胸部 X 线表现

          • 抗结核治疗有效

          • 临床可排除其他非结核性肺部疾病

          • PPD (5IU)强阳性，血清抗结核抗体阳性

          • 痰结核菌 PCR 和探针检测阳性

          • 肺外组织病理证实结核病变

        • 满足1项

          • 支气管肺泡灌洗( BAL)液中检出抗酸分枝杆菌

          • 支气管或肺部组织病理证实结核病变

    4. 4.

      记录方式

      1. 1.

        痰菌检查：涂(+)、涂(-)、培(+)、培(-）；(无痰)或(未查)

      2. 2.

        治疗状况：

        • 初治（任一）：

          • 尚未开始抗结核治疗的病人

          • 正进行标准化疗方案用药而未满疗程的病人

          • 不规则化疗未满1个月的病人

        • 复治（任一）：

          • 初治失败的病人

          • 规则用药满疗程后痰菌又复阳的病人

          • 不规则化疗超过 1 个月的病人

          • 慢性排菌病人

      3. 3.

        记录举例：继发型肺结核 双上　涂（＋）　复治

  8. 8.

    治疗

    1. 1.

      原则：早期、规律、全程、适量、联合

    2. 2.

      主要作用：①杀菌作用②防止耐药产生③灭菌

    3. 3.

      生物学机制：

      • 耐药性：联合用药，通过交叉杀菌作用防止耐药性产生

      • 间歇化学治疗：接触异烟肼（6～9日）和利福平（2～3日）24小时后可有延缓生长期

      • 顿服：效果优于分次口服

    4. 4.

      常用药物

      1. 1.

        异烟肼（INH, H）

        • 早期杀菌力最强者

        • A菌群（快速繁殖）

        • 不良反应：周围神经炎（可服用维生素B6--吡哆醇）

      2. 2.

        利福平（RFP, R）

        • 对C菌群（半静止）有独特杀菌作用

        • 不良反应：

          • 利福平及其代谢物为橘红色（大小便、眼泪）

          • 出现黄疸--立即停药

      3. 3.

        吡嗪酰胺（PZA, Z）

        • 具有独特的杀菌作用，主要是杀灭巨噬细胞内酸性环境中的B菌群（半静止）

        • 不良反应：高尿酸血症

      4. 4.

        链霉素（SM, S）

        • 具有独特的杀菌作用，主要是杀灭巨噬细胞外碱性环境中的结核分枝杆菌

        • 不良反应：耳毒性、前庭功能损害和肾毒性

      5. 5.

        乙胺丁醇（EMB，E）

        • 不良反应：视神经炎（儿童不用）

    5. 5.

      标准化学治疗方案


      		 每日用药
      		 间歇用药
      初治 涂（＋） 2HRZE/4HR 2H3R3Z3/4H3R3
      涂（－） 2HRZ/4HR 2H3R3Z3/4H3R3
      复治 涂（＋） 2HRZSE/4-6HRE 2H3R3Z3S3E3/6H3R3E3

      注：2H3R3Z3 异烟肼、利福平、吡嗪酰胺，每周3次，2个月

    6. 6.

      其他治疗

      1. 1.

        对症治疗

      2. 2.

        糖皮质激素：仅用于结核毒性症状严重者； 必须确保在有效抗结核药物治疗的情况下使用

      3. 3.

        肺结核外科手术治疗：

        • 经合理化学治疗后无效

        • 多重耐药的厚壁空洞

        • 大块干酷灶

        • 结核性脓胸

        • 支气管胸膜瘘

        • 大咯血保守治疗无效

  9. 9.

    相关疾病

    • HIV/AIDS：结核病是其最常见的机会感染性疾病

    • 肝炎：治疗药物多有潜在的肝毒性作用，用药前和用药过程中应定期监测肝功能

    • 糖尿病：肺结核的治疗必须在控制糖尿病的基础上才能奏效

    • 硅沉着病：其病人是并发肺结核的高危人群

  10. 10.

    控制策略与措施

    • 全程督导化学治疗（DOTS）

    • 病例报告和转诊：肺结核属乙类传染病

    • 病例登记和管理

    • 卡介苗接种

    • 预防性化学治疗
