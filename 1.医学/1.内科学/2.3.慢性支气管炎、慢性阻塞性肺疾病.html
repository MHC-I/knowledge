2.3.慢性支气管炎、慢性阻塞性肺疾病

  1. 1.

    慢性支气管炎

    • 慢性支气管炎( chronic bronchitis)：简称慢支，是气管、支气管黏膜及其周围组织的慢性非特异性炎症

    1. 1.

      病因

      1. 1.

        吸烟

      2. 2.

        职业粉尘和化学物质

      3. 3.

        空气污染

      4. 4.

        感染因素

    2. 2.

      病理

      1. 1.

        支气管上皮细胞变性、坏死、脱落，后期出现鳞状上皮化生，纤毛变短、粘连、倒伏、脱失

      2. 2.

        多种炎症细胞浸润（以中性粒细胞、淋巴细胞为主）

      3. 3.

        杯状细胞和黏液腺肥大增生、分泌旺盛，大量黏液潴留

      4. 4.

        支气管壁的损伤-修复过程反复发生 ，进而引起支气管结构重塑，胶原含量增加，瘢痕形成

    3. 3.

      临床表现： 缓慢起病，病程长，反复急性发作而使病情加重，急性加重的主要原因是呼吸道感染

      • 咳嗽：晨间咳嗽为主

      • 咳痰：白色黏液或浆液泡沫性痰，清晨排痰较多

      • 喘息或气急

    4. 4.

      相关检查

      1. 1.

        X 线检查：早期可无表现，后期纹理增粗、紊乱，呈网状或条索状、斑点状阴影，以双下肺明显

      2. 2.

        呼吸功能检查：使用支气管扩张剂后，第一秒用力呼气容积（FEV1）与用力肺活量（FVC）的比值（FEV1/FVC）< 70%提示已发展为慢性阻塞性肺疾病

      3. 3.

        血液检查：白细胞总数和（或）中性粒细胞计数增高

      4. 4.

        痰液检查：可培养出致病菌

    5. 5.

      诊断：依据咳嗽、咳痰或伴有喘息，每年发病持续 3 个月，连续 2 年或 2 年以上，并排除其他可以引起类似症状的慢性疾病

    6. 6.

      鉴别诊断

      1. 1.

        支气管哮喘：对抗生素无效，支气管激发试验阳性

      2. 2.

        嗜酸性粒细胞性支气管炎：诱导痰检查嗜酸性粒细胞比例增加( ≥3% )

      3. 3.

        肺结核

      4. 4.

        支气管肺癌

      5. 5.

        特发性肺纤维化：在胸部下后侧可闻及爆裂音（Velcro 啰音）

      6. 6.

        支气管扩张：反复大量咯腋痰或反复咯血

    7. 7.

      治疗

      1. 1.

        急性加重期的治疗

        • 控制感染

        • 镇咳祛痰

        • 平喘

      2. 2.

        缓解期治疗

        • 戒烟

        • 增强体质，预防感冒

        • 反复呼吸道感染者可试用免疫调节剂或中医中药

  2. 2.

    肺气肿

    • 肺气肿：是指肺部终末细支气管远端气腔出现异常持久的扩张，并伴有肺泡和细支气管的破坏，而无明显的肺纤维 化

  3. 3.

    慢性阻塞性肺疾病

    • 慢性阻塞性肺疾病(COPD)：简称慢阻肺，是一种常见的、可以预防和治疗的疾病，其特征是持续存在的呼吸系统症状和气流受限（进行性加重），通常与显著暴露于有害颗粒或气体引起的气道和(或)肺泡异常有关

    • 当慢性支气管炎、肺气肿病人肺功能检查出现持续气流受限时，能诊断为慢阻肺；其他疾病导致的持续气流受限则不属于慢阻肺

    1. 1.

      病因：与慢支炎相似

    2. 2.

      发病机制

      共同作用，最终产生两种重要病变：① 小气道病变；② 肺气肿病变，造成慢阻肺特征性的持续性气流受限

      1. 1.

        炎症机制：中性粒细胞最重要

      2. 2.

        蛋白酶-抗蛋白酶失衡机制：蛋白酶增多或抗蛋白酶（α1-AT）不足均可导致组织结构破坏，产生肺气肿

      3. 3.

        氧化应激机制：

        • 破坏大分子物质，破坏细胞外机制

        • 引起蛋白酶-抗蛋白酶失衡

        • 促进炎症反应

    3. 3.

      病理

      • 慢支炎

      • 肺气肿：肺过度膨胀，弹性减退；表面可见多个大小不一的肺大疱

    4. 4.

      病生：持续气流受限致肺通气功能障碍

      • 通气血流比例失调

        • 有通气，无血流：肺泡膨胀 ➡️ 肺毛细血管大量减少 ➡️ 肺泡壁无血液灌流 ➡️ 生理无效腔扩大

        • 有血流，无通气：部分肺区有血流灌注，但肺泡通气不良 ➡️ 功能性分流增加

      • 肺泡及毛细血管大量丧失，弥散面积减少 ➡️换气功能障碍

    5. 5.

      临床表现

      1. 1.

        慢支炎表现

      2. 咳嗽：晨间咳嗽为主

      3. 咳痰：白色黏液或浆液泡沫性痰，清晨排痰较多

      4. 喘息或气急

      5. 2.

        气短或呼吸困难：进行性加重，是慢阻肺的标志性症状

      6. 3.

        肺气肿体征

        • 视诊：桶装胸，呼吸变浅，频率增快，缩唇呼吸

        • 触诊：两侧语颤减弱

        • 叩诊：肺部过清音，心浊音界缩小，肺下界和肝浊音界下降

        • 听诊：两肺呼吸音减弱，呼气期延长，可闻及干、湿啰音

    6. 6.

      检查

      1. 1.

        肺功能检查

      2. 2.

        胸部 CT：高分辨率 CT 对辨别小叶中央型或全小叶型肺气肿以及确定肺大疱的大小和数量，有较高的敏感性和特异性

    7. 7.

      诊断：

      • 呼吸功能检查：吸人支气管扩张剂后，FEV1/FVC<70%

      • 同时排除其他已知病因或具有特征病理表现的气流受限疾病

    8. 8.

      稳定期病情严重程度评估

      1. 1.

        肺功能评估：GOLD

        前提--慢阻肺病人吸人支气管扩张剂后 FEV1/FVC<70%

      2. 2.

        症状评估：mMRC

      3. 3.

        急性加重风险评估：

        分组 特征 急性加重次数（风险） mMRC 分级（症状） 治疗
        A 低风险，症状少 急性加重 ≤1 0～1 短效 B\M，必要时
        B 低风险，症状多 急性加重 ≤1 ≥2 长效 B\M
        C 高风险，症状少 急性加重 ≥2，
        或因急性加重住院 ≥1
        		 0～1 LAMA
        LABA 加LAMA/ICS
        D 高风险，症状多 急性加重 ≥2，
        或因急性加重住院 ≥1
        		 ≥2 LAMA 加 LABA/ICS
    9. 9.

      鉴别诊断

      • 哮喘：慢阻肺多为中年发病，症状缓慢进展，哮喘多为儿童或青少年期起病，症状起伏大，常伴有过敏史、鼻炎和(或)湿疹等，部分病人有哮喘家族史

    10. 10.

      治疗

      1. 1.

        稳定期治疗

        • 教育与管理：劝导吸烟的病人戒烟，这是减慢肺功能损害最有效的措施

        • 支气管扩张剂：现有控制症状的主要措施

          • β2 肾上腺素受体激动剂：短效（SABA）、长效（LABA）

          • 抗胆碱药：短效（SAMA）、长效（LAMA）

          • 茶碱类

        • 吸入糖皮质激素（ICS）

        • 祛痰药：盐酸氨溴索

        • 长期家庭氧疗 （LTOT）：一般用鼻导管吸氧，氧流量为 1.0~2.0L/min，吸氧时间 >15h/d（持续低流量吸氧）

      2. 2.

        急性加重期治疗

        • 确定急性加重的原因：最多见的原因是细菌或病毒感染

        • 支气管扩张剂：严重者大剂量雾化吸入治疗

        • 低流量吸氧：应避免吸人氧浓度过高引起二氧化碳潴留

        • 抗生素

        • 糖皮质激素：连续 5～7 天

        • 机械通气

      3. 3.

        外科治疗：肺大疱切除术、肺减容手术、肺移植
