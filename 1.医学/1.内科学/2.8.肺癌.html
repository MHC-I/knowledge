2.8.肺癌

肺癌：起源于呼吸上皮细胞（支气管、细支气管和肺泡）的恶性肿瘤，是最常见的肺部原发性恶性肿瘤

  1. 1.

    流行病学：死亡率列首位，男性发病率在所有癌症中列首位，女性发病率仅次于乳腺癌列第二位

  2. 2.

    病因：

    • 吸烟

    • 职业致癌因子

    • 空气污染

    • 电离辐射

    • 饮食与体力活动

    • 遗传和基因改变

  3. 3.

    分类

    1. 1.

      解剖学分类

      • 中央型肺癌：发生在段及以上支气管的肺癌，以鳞癌和小细胞癌多见

      • 周围型肺癌：发生在段支气管以下的肺癌，以腺癌多见

    2. 2.

      组织病理学分类

      1. 1.

        非小细胞肺癌（NSCLC)

        1. 1.

          鳞癌

          • 早期常引起支气管狭窄

          • 癌组织易变性、坏死，形成空洞或癌性肺脓肿

          • 生长较慢，转移晚，手术切除机会较多

        2. 2.

          腺癌

          • 肺癌最常见的类型

          • 女性多见

          • 富含血管，局部浸润和血行转移较早，易累及胸膜引起胸腔积液

        3. 3.

          大细胞癌

          • 未分化的非小细胞癌

          • 诊断不适用小活检和细胞学标本

          • 转移较晚，于术切除机会较大

      2. 2.

        小细胞癌（SCLC）

        • 以增殖快速和早期广泛转移为特征

        • 低分化（恶性程度高）的神经内分泌肿瘤

        • 具有内分泌和化学受体功能，可引起类癌综合征

        • 对化疗和放疗较敏感

  4. 4.

    临床表现

    1. 1.

      原发肿瘤引起的

      1. 1.

        咳嗽：无痰或少痰的剌激性干咳，呈高调金属音性咳嗽或刺激性呛咳。

      2. 2.

        痰血或咯血

      3. 3.

        气短或喘鸣

      4. 4.

        胸痛

      5. 5.

        发热：多由肿瘤引起的阻塞性肺炎所致，抗生素治疗效果不佳

    2. 2.

      肿瘤局部扩展引起的

      1. 1.

        胸痛：肿瘤侵犯胸膜或胸壁时，产生不规则的钝痛、隐痛或剧痛，在呼吸、咳嗽时加重

      2. 2.

        声音嘶哑：肿瘤压迫喉返神经(多见左侧 )使声带麻痹

      3. 3.

        吞咽因难：肿瘤侵犯或压迫食管

      4. 4.

        胸腔积液：累及胸膜或肺淋巴回流受阻

      5. 5.

        心包积液

      6. 6.

        上腔静脉阻塞综合征：肿瘤直接侵犯纵隔；或转移的肿大淋巴结压迫上腔静脉；或腔静脉内癌栓阻塞，

        引起静脉回流受阻 。 表现上肢、颈面部水肿和胸壁静脉曲张

      7. 7.

        Horner综合征：肺上沟瘤（Pancoast tumor）压迫颈交感神经，引起病侧上睑下垂、瞳孔缩小、眼球内陷，同侧额部与胸壁少汗或无汗

    3. 3.

      肿瘤远处转移引起的：肺癌可转移至任何器官系统

      1. 1.

        中枢神经系统转移

      2. 2.

        骨髓转移

      3. 3.

        腹部转移

      4. 4.

        淋巴结转移：锁骨上窝淋巴结最常见（同胃癌）

    4. 4.

      副癌综合征（以SCLC多见）

      1. 1.

        内分泌综合征

        • 抗利尿激素分泌异常综合征（SIADH）

        • 异位 ACTH 综合征：表现为库欣综合征

        • 高钙血症

      2. 2.

        骨髓-结缔组织综合征

        • 原发性肥大性骨关节病：病人有杵状指(趾)，多为NSCLC

        • 神经-肌病综合征：肌无力样综合征

      3. 3.

        血液学异常

  5. 5.

    相关检查

    1. 1.

      影像学检查

      1. 1.

        X线胸片：最常用

        1. 1.

          中央型肺癌

          • 一侧肺门类圆形阴影，边缘毛糙，可有分叶或切迹

          • 下缘可表现为倒S状影像

          • 间接：局限性肺气肿、肺不张征象

        2. 2.

          周围型肺癌

          • 结节 、球状、网状阴影或磨玻璃影

          • 边缘常呈分叶状，伴有脐凹征或细毛刺，常有胸膜牵拉

          • 肿瘤向肺门淋巴结转移，可见引流淋巴管增粗成条索状阴影伴肺门淋巴结增大

          • 癌组织坏死与支气管相通后，表现为厚壁 ，偏心，内缘凹凸不平的癌性空洞

      2. 2.

        CT：确定部位；具有更高的分辨率，可发现肺微小病变和普通 X 线胸片难以显示的部位

      3. 3.

        MRI：明确肿瘤与大血管之间的关系、发现脑实质或脑膜转移

      4. 4.

        PET-CT：对发现早期肺癌和其他部位的转移灶，以及肿瘤分期与疗效评价均优于任何现有的其他影像学检查

    2. 2.

      病理学检查

      1. 1.

        痰脱落细胞学检查：必须获得气道深部的痰液，及时送检，至少送检3次以上，不推荐作为确诊依据

      2. 2.

        支气管镜：主要方法

    3. 3.

      肿瘤标志物检测

  6. 6.

    鉴别诊断

    1. 1.

      肺结核：抗结核治疗有效； 粟粒型肺结核X线影像表现为细小、分布均匀、密度较淡的粟粒样结节病灶

    2. 2.

      肺炎：抗生素治疗有效

    3. 3.

      肺脓肿：影像学可见均匀的大片状阴影，空洞内常见液平

  7. 7.

    治疗

    • 原则：多学科综合治疗模式（MDT），强调个体化治疗


      		 NSCLS SCLC
      手术治疗 根治性手术切除是首选治疗手段 90%以上就诊时已有胸内或远处转移,一般不推荐手术治疗
      药物治疗 对化疗的反应较差
      一线化疗推荐含铂两药（顺铂、卡铂）联合化疗
      一线治疗失败者，推荐多西他赛或培美曲赛单药二线化疗
      靶向治疗主要应用于非小细胞肺癌中的腺癌病人
      		 对化疗非常敏感，是治疗的基本方案
      一线化疗药物包括依托泊苷或伊立替康联合
      顺铂或卡铂，共4～6 个周期
      放射治疗 敏感性：SCLC>鳞癌>腺癌

    • 放射治疗

      • 根治性放疗用于病灶局限、因解剖原因不便手术或其他原因不能手术者

      • 姑息性放疗的目的在于抑制肿瘤的发展

      • 辅助放疗适用于术前放疗、术后切缘阳性的病人

      • 术前化疗(新辅助化疗)可使原先不能手术的病人降低TNM分期而可以手术

      • 预防性放疗适用于全身治疗有效的小细胞肺癌病人全脑放疗

  8. 8.

    预后：取决于早发现、早诊断、早治疗
