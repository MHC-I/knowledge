2.6.肺部感染性疾病

肺炎

一、概述

肺炎( pneumonia)：指终末气道、肺泡和肺间质的炎症，可由病原微生物 、理化因素、免疫损伤、过敏及药物所致，细菌性肺炎最常见

  1. 1.

    发病机制

    1. 1.

      宿主因素

      • 支气管内黏液—纤毛运载系统完整性

      • 肺泡巨噬细胞等防御的完整性

    2. 2.

      病原体因素(途径)

      • 空气吸入

      • 血行播散

      • 邻近感染部位蔓延

      • 上呼吸道定植菌的误吸

  2. 2.

    分类

    1. 1.

      解剖分类

      1. 1.

        大叶性(肺泡性)肺炎：

        • 病原体经肺泡间孔(Cohn 孔)向其他肺泡扩散

        • 部分肺段或整个肺段、肺叶发生炎症

        • 通常不累及支气管

        • X线显示肺叶或肺段的实变阴影

        • 常见致病菌：肺炎链球菌

      2. 2.

        小叶性(支气管性)肺炎病原体经支气管人侵

        • 病原体经支气管入侵

        • X线影像显示为沿着肺纹理分布的不规则斑片状阴影，边缘密度浅而模糊，无实变征象 ，肺下叶常受累

      3. 3.

        间质性肺炎

    2. 2.

      患病环境分类

      1. 1.

        社区获得性肺炎(CAP)

        • CAP：是指在医院外罹患的感染性肺实质(含肺泡壁，即广义上的肺间质)炎症，包括具有明确潜伏期的病原体感染在入院后于潜伏期内发病的肺炎

        • 诊断依据(临床表现)：符合①②及③中任何一项

          • 社区发病

          • 胸部影像学检查显示片状、斑片状浸润性阴影或间质性改变

          • 肺炎相关临床表现:

            • 新近出现的咳嗽、咳痰或原有呼吸道疾病症状加重并出现脓性痰

            • 发热

            • 肺实变体征和(或)闻及湿性啰音

            • WBC或

          • 最常见病原体：肺炎链球菌

      2. 2.

        医院获得性肺炎(HAP)

        • HAP：指病人住院期间没有接受有创机械通气，未处于病原感染的潜伏期，且入院 ≥ 48 小时后在医院内新发生的肺炎

          • 呼吸机相关性肺炎(VAP)：是指气管插管或气管切开病人，接受机械通气 48 小时后发生的肺炎及机械通气撤机、拔管后 48 小时内出现的肺炎

        • 诊断依据(临床表现)：

          胸部 X 线或 CT 显示新出现或进展性的浸润影、实变影、 磨玻璃影，外加下列两个及以上症状：

          • 发热，体温 >38°C

          • 脓性气道分泌物

          • 外周白细胞 或

        • 最常见病原体：鲍曼不动杆菌

  3. 3.

    临床表现

    • 大多有发热

    • 肺实变时语颤增强，并发胸腔积液时语颤减弱

  4. 4.

    诊断

    1. 1.

      确定肺炎诊断

    2. 2.

      评估严重程度

      • 严重性决定于三个主要因素:

        • 肺部局部炎症程度

        • 肺部炎症的播散程度

        • 全身炎症反应程度

      • 是否需要住院治疗(CURB-65)：满足1项得1分：0~1分门诊治疗，2分院外治疗，3～5分住院治疗

        • 意识障碍

        • 尿素氮>7mmol/L

        • 呼吸频率注30 次/分

        • 收缩压<90mmHg或舒张压≥60mmHg

        • 年龄≥65岁

      • 重症肺炎：符合1 项主要标准或≥3项次要标准

        • 主要标准：

          • 需要气管插管行机械通气治疗

          • 脓毒症体克经积极液体复苏后仍需要血管活性药物治疗

        • 次要标准：

          • 呼吸频率≥30 次/分

          • PaO2/FiO2(氧合指数)≤250mmHg

          • 多肺叶浸润

          • 意识障碍和(或)定向障碍

          • 血尿素氮≥20mg/dl

          • 收据压<90mmHg，需要积极的液体复苏

    3. 3.

      确定病原体

      1. 1.

        痰：浓度时，可以认为是肺部感染的致病菌

      2. 2.

        经支气管镜或人工气道吸引:浓度，……

      3. 3.

        防污染样本毛刷：浓度，……

      4. 4.

        支气管肺泡灌洗：浓度，……

  5. 5.

    治疗

    • 抗感染治疗是关键环节，包括经验性治疗(最常用)和抗病原体治疗

    • 重症肺炎首先应选择广谱的强力抗菌药物，并应足量、联合用药

    • 在初始治疗后72小时对病情进行评价

二、细菌性肺炎

肺炎链球菌肺炎

  1. 1.

    发病机制

    • 肺炎链球菌(SP)，G+，是寄居在口腔及鼻咽部的一种正常菌群

    • SP不产生毒素，不引起组织坏死或形成空洞(整个病程没有肺泡壁和其他结构的损伤)。 其致病力是由于高分子多糖体的荚膜对组织的侵袭作用

    • 含菌的渗出液经Cohn孔向肺的中央部分扩展(大叶性)

    • 病变可引起通气/血流比例失调，导致缺氧

  2. 2.

    病理

    • 铁锈色痰和红色肝变期有关

  3. 3.

    临床表现

    1. 1.

      易感人群：健康的青壮年或老年与婴幼儿，男性较多见

    2. 2.

      症状：

      • 发病前常有受凉、淋雨、疲劳、醉酒、病毒感染史，多有上呼吸道感染的前驱症状

      • 痰少，可带血或呈铁锈色

    3. 3.

      体征：口角及鼻周有单纯疱疹

    4. 4.

      病程：自然病程大致1～2周，使用有效的抗菌药物后体温1～3天恢复正常

  4. 4.

    治疗

    1. 1.

      抗菌药物治疗：首选青霉素

    2. 2.

      支持疗法：不用解热药，以免过度出汗、脱水及干扰真实热型，导致临床判断错误

    3. 3.

      并发症的处理：体温降而复升或3天后仍不降者，应考虑SP的肺外感染

葡萄球菌肺炎

  1. 1.

    发病机制

    由葡萄球菌引起的急性肺化脓性炎症，致病物质主要是毒素与酶(金葡菌凝固酶为阳性，是化脓性感染的主要原因)

  2. 2.

    病理

    1. 1.

      经呼吸道吸人的肺炎常呈大叶性分布或广泛的融合性的支气管肺炎

    2. 2.

      皮肤感染灶中的葡萄球菌可经血液循环抵达肺部形成单个或多发性肺脓肿

  3. 3.

    临床表现

    • 症状：

      • 起病多急骤，寒战、高热，体温多高达到39～40°C

      • 痰脓性，量多，带血丝或呈脓血状(砖红色胶冻样痰)

    • 体征：常与严重的中毒症状和呼吸道症状不平行

  4. 4.

    实验室检查

    1. 1.

      白细胞计数明显升高，中性粒细胞比例增加，核左移

    2. 2.

      X线：①单个或多发的液气囊腔；②X线影像阴影的易变性

  5. 5.

    治疗：

    • 早期清除和引流原发病灶，选用敏感的抗菌药物

    • 耐药率高，选用耐青霉素酶的半合成青霉素或头孢菌素

    • 对于MRSA，则应选用万古霉素、替考拉宁和利奈唑胺

三、其他病原体肺炎

肺炎支原体肺炎

MP是介于细菌和病毒之间、兼性厌氧、能独立生活的最小微生物

  1. 1.

    发病机制：

    • 释放毒性代谢产物引起支气管、细支气管黏膜层破坏， 纤毛运动减弱

    • 产生特异性抗体，使原有症状加重

  2. 2.

    临床表现

    • 起病缓慢

    • 咳嗽明显，多为发作性干咳，夜间加重，持久的阵发性剧咳为典型表现

    • 可伴有鼻咽部和耳部的疼痛

  3. 3.

    实验室检查：

    • 起病2周后，病人冷凝集试验阳性， 滴度注≥1:32，如果滴度逐步升高，更有诊断价值

    • 血清支原体IgM抗体≥1:64，或恢复期抗体滴度有4倍增高，可进一步确诊

    • X线检查显示肺部多种形态的浸润影，呈节段性分布，以肺下野为多见

  4. 4.

    治疗

    • 有自限性，多数病例不经治疗可自愈

    • 首选大环内酯类抗生素(无细胞壁)，不敏感者可选用呼吸喹诺酮类

肺脓肿

肺脓肿(lung abscess)：是由多种病原体所引起的肺组织化脓性病变，早期为化脓性肺炎，继而坏死、液化，脓肿形成

  1. 1.

    发病机制

    1. 1.

      吸入性肺脓肿

      1. 1.

        病原体经口、鼻、咽腔吸入致病

      2. 2.

        诱因：意识障碍（如麻醉、醉酒），受寒 、极度疲劳

      3. 3.

        脓肿常为单发，吸入物易进入右肺

    2. 2.

      继发性肺脓肿

      1. 1.

        继发于细菌性肺炎

      2. 2.

        支气管扩张、支气管囊肿、支气管肺癌、肺结核空洞等继发感染

      3. 3.

        肺部邻近器官化脓性病变

      4. 4.

        支气管异物阻塞：是导致小儿肺脓肿的重要因素

    3. 3.

      血源性肺脓肿：外伤感染、疖、痈等的菌栓经血行播散到肺

  2. 2.

    病理

    感染物阻塞细支气管，致病菌繁殖引起小血管炎性栓塞，肺组织化脓性炎症、坏死，形成肺脓肿， 继而坏死组织液化破溃到支气管。炎症迁延 3 个月以 上则称为慢性肺脓肿

  3. 3.

    临床表现

    1. 1.

      早期为肺炎症状，体温可高达39～40°C

    2. 2.

      1 2周后，咳嗽加剧，咳出大量脓臭痰，每日可达300500ml，体温旋即下降

    3. 3.

      慢性肺脓肿：慢性咳嗽、咳脓痰、反复咯血、不规则发热等，常呈贫血、消瘦等慢性消耗病态

  4. 4.

    实验室检查

    1. 1.

      生化检查：细胞总数达(20~30)x，核左移明显，常有毒性颗粒

    2. 2.

      微生物检查

    3. 3.

      影像学检查：

      1. 1.

        大片浓密模糊炎性浸润阴影，边缘不清

      2. 2.

        薄壁囊性空洞

      3. 3.

        纵隔向患侧移位，健侧发生代偿性肺气肿

      4. 4.

        并发脓胸者，患侧胸部呈大片浓密阴影；若伴发气胸则可见液平面

    4. 4.

      纤维支气管镜检查

  5. 5.

    诊断

    • 昏迷呕吐、异物吸入，急性发作的畏寒、高热、咳嗽和咳大量脓臭痰等病史

    • 白细胞总数和中性粒细胞显著增高

    • 肺野大片浓密炎性阴影中有脓腔及液平面的X线征象

  6. 6.

    治疗

    1. 1.

      抗生素治疗：青霉素与甲硝唑合用

    2. 2.

      脓液引流：

      1. 1.

        药物引流：祛痰药雾化吸入

      2. 2.

        体位引流：应使脓肿处于最高位

      3. 3.

        纤维支气管镜冲洗：有明显痰液阻塞征象时可选

    3. 3.

      手术治疗，适应征：

      1. 1.

        肺脓肿病程超过3个月，经内科治疗脓腔不缩小，或脓腔过大（5cm以上）估计不易闭合者

      2. 2.

        大咯血经内科治疗无效或危及生命

      3. 3.

        伴有支气管胸膜瘘或脓胸经抽吸、引流和冲洗疗效不佳者

      4. 4.

        支气管阻塞限制了气道引流
